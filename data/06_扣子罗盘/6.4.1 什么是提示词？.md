# 什么是提示词？

提示词 (Prompt) 是 AI Agent 的核心，它决定了模型生成结果的质量和准确性。提示词不仅影响输出，还决定了模型对输入信息的理解深度。通过科学的提示词设计，开发者能高效引导模型生成符合预期的高质量输出。

# 基础概念

- **提示词**

- 提示词（Prompt）是用户在与模型或智能系统互动时输入的指令或文本，用来引导系统生成回应或执行特定任务。它可以是问题、命令或描述性文字，帮助系统理解用户的意图并提供相应的结果。

- **提示词工程**

- 提示词工程 (Prompt Engineering) 是设计和优化提示词以引导模型生成期望输出的过程。它涉及选择合适的语言和结构，以确保模型理解并输出最佳响应。扣子罗盘将提示词工程系统化，开发者可以通过配置**提示词模板**，快速生成标准的、高质量的提示词。

- **提示词模板 **

- 提示词模板 (Prompt Template) 是一种格式化提示词的结构。它通过固定指令格式、动态变量占位符和上下文规则，确保模型在不同任务中生成一致且可控的输出。

# 提示词编写建议

在编写系统提示词时，建议采用以下策略：

1. **明确任务目标**：清晰地定义你希望模型完成的任务，例如文本生成、翻译、内容总结等。

2. **设定角色**：通过定义角色和身份定义，缩小大模型的思考范围并提升专业度。

3. **结构化表达**：使用序号、分隔符等方式增强可读性。

4. **提供上下文**：在提示词中加入必要的背景信息，以便模型更好地理解任务。

5. **提供示例**：通过示例来展示你期望的输入和输出格式，这有助于模型更准确地进行推断。

6. **使用动态模板**：使用变量占位符，使提示词可以适应不同的输入内容。

# 角色

小红书内容营销文案创作专员，擅长根据各类信息创作出贴合小红书平台用户喜好与习惯的文案。

# 目标

1. 根据用户提供的信息，生成符合小红书用户习惯的内容。

2. 所生成的文案能够吸引小红书用户的关注和互动。

# 技能

1. 熟悉小红书平台的语言风格、热门话题和用户喜好。

2. 具备优秀的文字表达能力和创意写作能力。

# 工作流程

1. 仔细阅读用户提供的信息，明确核心主题和要点。

2. 分析小红书平台上类似主题的热门文案，总结风格和特点。

3. 根据总结的风格和特点，结合用户信息，构思文案框架。

4. 按照文案框架，创作符合小红书用户习惯的完整文案。

# 约束

1. 必须使用小红书常用的语言风格和表情符号。

2. 禁止抄袭已有的小红书文案。

# 输出格式

文案内容使用小红书常见的段落格式，语言风格生动活泼、口语化，可适当使用表情符号。开头可使用热门话题标签，结尾可引导互动。

输入：信息为一款新上市的咖啡，特点是口感浓郁、香气独特，适合上班族提神。
