# 终端用户用量查询和配额管控

本文介绍如何在 API Playground 页面查看终端用户的用量数据、用量明细、管理设备配额以及查看设备流水详情。以便开发者清晰掌握终端用户资源使用状况，从而实现精细化管理。

> [!NOTE]
>
> **说明**
>
> - **套餐限制**：企业旗舰版、原企业版。
>
> - **角色限制**：企业超级管理员或管理员。
>
> - **数据更新**：设备管理页面的数据将在次日更新，暂不支持查看当天的使用数据。

## 设备概览

通过**设备概****览**页面，你可以快速了解设备的日活数、对话次数、语音转文字用量、文字转语音用量、实时音视频用量等关键指标，从而分析设备使用情况并优化资源配置。

1. 企业超级管理员或管理员登录[扣子编程](https://code.coze.cn/home)。

2. 在左下角单击个人头像，选择企业> **团队与企业管理**。

3. 在左侧导航栏中选择**设备管理**页签，在**设备概览**页面，在顶部选择统计维度和时间段，支持查看近 7 天、近 30 天和近 60 天的数据。

- 统计维度默认为按设备维度进行统计，你也可以选择按自定义维度。如果选择按自定义维度，需要先在 OAuth JWT 授权的 Payload 中设置 custom\_consumer 参数，用于设置用量统计的自定义维度，具体请参见 [OAuth JWT 授权（开发者）](https://docs.coze.cn/developer_guides/oauth_jwt)。

## 查询用量明细

在**团队与企业管理**的**设备管理**页签中，单击**用量明细及管控**页签，可以查看各个设备最近 1 天或最近 7 天的用量明细。

<img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/ac18918cfc96433691d98edf3eda11db~tplv-goo7wpa0wc-quality:q75.awebp" width="600">

明细列表中的参数说明如下：

| **参数** | **说明** |
| --- | --- |
| 设备 ID | 用于唯一标识每台设备。 |
| 累计对话次数 | 设备自使用以来的总对话次数。 |
| 当期对话次数 | 设备在选定时间段内的对话次数。 |
| 累计账单金额 | 设备自使用以来的总费用。 |
| 当期账单金额 | 设备在选定时间段内的费用。 |

## 管理设备配额

你可以为每台设备设置免费使用额度，以便用户在初始阶段免费体验设备功能，同时避免资源的过度使用。通过合理的额度配置，既能满足用户的试用需求，又能有效控制资源消耗。

> [!NOTE]
>
> **说明**
>
> - 未配置限额时，默认支持无限使用。
>
> - 任何一个配额耗尽时，设备将无法继续使用。

在**用量明细及管控**页面，单击右上角的**全局配额（设备维度）**，可以配置设备的配额，对你所在火山引擎主账号下的所有设备生效，支持按积分维度或语音通话时长维度设置配额。

<img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/65a7a5f77f6648409ca9fd5fbc76e4f4~tplv-goo7wpa0wc-quality:q75.awebp" width="400">

配置示例：

例如你可以设置每个设备累计额度为 5000 积分，每天可用 500 积分。当设备 A 在某一天的使用积分达到 500 上限后，设备 A 当天将无法继续使用，需等到次日才能恢复使用，当设备 A 的累计使用积分达到 5000 上限后，设备 A 将无法继续使用。

| **维度** | **参数** | **说明** |

## 查看设备流水详情

通过查看设备流水详情，你可以详细了解设备的费用构成和资源使用情况，便于精细化管理。

在**用量明细及管控**页面，单击对应设备右侧操作列的**流水**，可以查看该设备在指定时间范围内，每天的用量流水详情，包括日期、计费项、用量和金额。计费项包括：RTC 实时音视频费用、ASR 语音合成费用、TTS 语音识别费用、大模型费用。
