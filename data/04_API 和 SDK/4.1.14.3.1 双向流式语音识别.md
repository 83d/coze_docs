# 双向流式语音识别

扣子编程提供流式语音识别 WebSocket OpenAPI，可以将指定的音频判断转为文字，支持识别中英文双语种。

双向流式语音识别场景下的各类事件详细信息可参考[双向流式语音识别事件](https://docs.coze.cn/developer_guides/asr_event)。

## 接口信息

| **URL** | wss://ws.coze.cn/v1/audio/transcriptions |
| --- | --- |
| **Headers** | Authorization Bearer *$Access\_Token*  用于验证客户端身份的访问令牌。你可以在扣子编程中生成访问令牌，详细信息请参考[准备工作](https://www.coze.cn/docs/developer_guides/preparation)。 |
| **权限** | createTranscription |
| **接口说明** | 将指定的音频判断转为文字，支持识别中英文双语种。 |

## 建连示例代码

ws module(Node.js)

websocket-client(Python)

Websocket(browsers)

import WebSocket from 'ws';

const url = `wss://ws.coze.cn/v1/audio/transcriptions?authorization=Bearer ${ACCESS\_TOKEN}`;

const ws = new WebSocket(url);

ws.on('open', function open() {

console.log('Connected to server.');

});

ws.on('message', function incoming(*message*) {

console.log(JSON.parse(message.toString()));

});
