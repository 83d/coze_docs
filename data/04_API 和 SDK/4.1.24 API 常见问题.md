# API 常见问题

## 调用非流式接口时，返回结果中为什么没有数据？

- **流式 API**：调用时会在返回结果中直接包含模型生成的内容。

- **非流式 API**：响应消息中**不会即时返回模型生成的内容**，需通过后续 API 查询结果。例如：调用[发起对话](https://docs.coze.cn/developer_guides/chat_v3) API 时，如果设置 stream=false，你需要通过如下方法查询模型生成的内容：

1. 记录[发起对话](https://docs.coze.cn/developer_guides/chat_v3) API 响应中返回的 chat\_id 和 conversation\_id。

2. 调用[查看对话详情](https://docs.coze.cn/developer_guides/retrieve_chat) API 确认本次对话处理已结束。

3. 调用[查看对话消息详情](https://docs.coze.cn/developer_guides/list_chat_messages) API 查看模型回复的内容。

## 调用发起对话 API 时，工作流能异步运行吗？

调用[发起对话](https://docs.coze.cn/developer_guides/chat_v3) API 可与智能体进行对话，如果智能体中绑定了工作流，调用该 API 运行智能体内的工作流时，工作流将以同步方式运行，且不支持配置为异步运行。
