# 企业内部权限管理

本文围绕企业超级管理员规划、组织和工作空间划分、成员用量管控、分账账单查询等维度，介绍企业内部权限管理的最佳实践。

## 场景说明

AI 智能体开发已成为企业提升效率、优化服务的重要手段。对于设有多个部门的企业，不同部门在 AI 智能体开发和使用上有着不同的需求。当多部门协同使用扣子编程时，搭建清晰的组织架构、明确人员管理权限，是确保平台高效有序运行的关键。

本教程以某互联网企业的实际应用为案例。该企业为满足产品研发、业务运营、客户服务等核心部门的 AI 智能体开发需求，采购了扣子编程企业旗舰版，并以部门为单位，划分了组织架构和人员管理。

- **设定企业超级管理员**：由企业 IT 管理部门负责人作为主责任人，负责采购企业旗舰版套餐，并自动成为企业超级管理员。同时，为了协助负责人的日常事务处理，将部门助理也设定为超级管理员，负责企业旗舰版的日常工作，例如创建/添加成员、设置成员用量额度等。

- **搭建组织与工作空间**：

- 企业依据产品研发部、业务运营部、客户服务部等实际部门创建组织，确保各部门资源独立管理。

- 在组织内结合部门业务需求（如不同项目组、不同业务模块），规划专属工作空间，实现同一部门内不同业务单元的资源隔离与管控。

- 由各部门负责人担任组织超级管理员；各部门助理担任组织管理员，处理创建工作空间、添加/移除组织成员等日常事务。

- **积分用量管控**：企业超级管理员统一设置企业成员的积分用量。

- **查看分账账单**：按照组织、工作空间两个维度，清晰查看积分用量及对应分拆费用，助力企业把控各部门、各业务的平台使用成本。

## 设定企业超级管理员

购买企业旗舰版时，系统自动将火山引擎主账号（RootUser）设为超级管理员。同时，为满足企业日常管理需求，还支持自行添加一位企业超级管理员。

> [!NOTE]
>
> **说明**
>
> 系统默认的超级管理员，不支持转让。

从企业管理实践出发，考虑到 IT 管理部门对企业数据安全、合规要求具备全局把控能力，以及其在网络配置、系统权限管理等方面的专业能力。因此，指定 IT 管理部门负责人采购企业旗舰版，成为企业超级管理员 A。同时，设定部门助理担任另一位超级管理员 B，协助处理扣子编程的日常事务。

| **扣子角色** | **企业角色** | **核心职责** |
| --- | --- | --- |
| 企业超级管理员 A | IT 管理部门负责人 | 具备扣子账号最高级别权限，主要职责如下：  - 添加企业管理员。  - 管理企业套餐。  - 增购企业成员数。具体操作，请参考[购买企业版](https://docs.coze.cn/coze_pro/enterprise_plan)。  - 增购声音复刻-音色数量等音视频相关资源。具体操作，请参考[音视频费用](https://docs.coze.cn/coze_pro/asr_tts_fee)。  - 购买积分。具体操作，请参考[购买积分](https://docs.coze.cn/coze_pro/credits)。  - 办理套餐续费，避免因套餐到期影响企业正常使用。具体操作，请参考[续费订阅套餐](https://docs.coze.cn/coze_pro/premium_package)。 |
| 企业超级管理员 B | IT 管理部门助理 | 具备较高的扣子账号操作权限，协助系统超级管理员处理日常事务。主要职责如下：  - 创建、添加、移除企业成员。具体操作，请参考[添加员工到企业](https://docs.coze.cn/guides/create_member)。  - 创建企业组织并指定各个部门的负责人为组织的超级管理员。具体操作，请参考[创建组织](https://docs.coze.cn/guides/manage_organization)。  - 设置成员积分用量额度。具体操作，请参考[设置成员积分额度限制](https://docs.coze.cn/coze_pro/bills_and_usage)。  - 配置单点登录（SSO）、私有网络配置、数据加密管理等企业安全特性维护。具体操作，请参考[配置单点登录（SSO）](https://docs.coze.cn/guides/configure_sso)、[管理私网连接](https://docs.coze.cn/guides/private_link)、[数据加密管理](https://docs.coze.cn/guides/data_encryption_management)。 |

## 划定组织和工作空间

企业旗舰版支持多组织管理，企业可依据核心业务部门划分独立组织，每个组织独立设置对应的超级管理员和管理员，实现多部门的独立运作与统一管理。关于组织的详细介绍，请参考[创建组织](https://docs.coze.cn/guides/manage_organization)。

工作空间是资源组织的基础单元，不同空间内的资源和数据相互隔离。将企业成员添加到对应的工作空间后，成员可以进行业务操作，搭建智能体、工作流、应用等。关于空间的详细介绍，请参考[工作空间概述](https://docs.coze.cn/guides/teams)。

为满足各部门的业务需求，结合部门职能与项目特点，该互联网企业对组织与工作空间进行了如下规划：

| **组织名称** | **工作空间** | **空间所有者** | **组织超级管理员** | **组织与工作空间用途** |
| --- | --- | --- | --- | --- |
| 产品研发部 | 研发通用工作空间 | 项目负责人 | 研发部负责人 | 面向研发内部 AI 智能体的开发 |
|  | 电商项目工作空间 | 项目负责人 |  | 面向电商项目 AI 智能体的开发 |
|  | 教育项目工作空间 | 项目负责人 |  | 面向教育项目 AI 智能体的开发 |
| 业务运营部 | 产品运营工作空间 | 运营部负责人 | 运营部负责人 | 面向运营相关智能体的开发 |
| 客户服务部 | 客户服务工作空间 | 客服部负责人 | 客服部负责人 | 面向客服相关智能体的开发 |
| 默认组织 | 企业通用工作空间 | IT 管理部门负责人 | IT 管理部门负责人 | 面向企业通用智能体的开发 |

## 积分用量管控

为避免企业成员过度消耗积分，保障积分的合理分配，企业超级管理员可以为企业成员设置积分使用额度，实现精细化用量管控。

企业旗舰版支持设置累计额度限制和每月额度限制。

- 累计额度限制：开启额度限制后，当该成员的积分使用量达到额度上限值时，将无法继续使用需要抵扣积分的扣子编程功能。

- 每月额度限制：开启额度限制后，当该成员在当月的积分使用量达到额度上限值时，当月将无法继续使用需要抵扣积分的扣子功能。

在本互联网企业场景中，考虑到企业管理员与普通成员的工作内容和资源需求差异，按照角色维度设置每月的积分额度。具体操作，请参考[设置成员积分额度限制](https://docs.coze.cn/coze_pro/bills_and_usage)。

| **成员类型** | **月额度** |
| --- | --- |
| 企业管理员 | 5000 积分 |
| 企业成员 | 10000 积分 |

## 查看账单

开启分账功能后，系统将支持根据组织、工作空间维度查看积分用量和对应的分拆费用，助力企业成本核算与归属管理。企业可以通过拆分项标识[org:\*\*\*\*]-[ws:\*\*\*\*]，快速定位某组织下某工作空间内各个计费功能消耗的积分。详情请参考[查看分账账单](https://docs.coze.cn/coze_pro/bills_and_usage)。

- [org:\*\*\*\*]：组织名称或组织 ID

- [ws:\*\*\*\*]：工作空间名称或工作空间 ID
