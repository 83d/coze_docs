# OpenClaw 集成企业微信

OpenClaw 是一款开源的主动型 AI Agent。你可以通过 OpenClaw 将多渠道通信能力与大语言模型深度集成，创建拥有持久记忆与主动执行能力的定制化 AI 助理。

关于 OpenClaw 的详细说明，可参考[什么是 OpenClaw](https://docs.coze.cn/tutorial/openclaw)。

## 准备工作

| **项目** | **说明** |
| --- | --- |
| 版本限制 | 仅**个人高阶版**、**个人旗舰版**、**企业标准版**、**企业旗舰版**用户可创建和部署。 |
| 项目配额 | 每个用户最多拥有 **1 个** OpenClaw 项目，你可以删除旧项目以释放额度。 |
| 使用限制 | 如果取消部署，对话随时可能因沙箱回收而中断。 |
| 费用说明 | - 使用 OpenClaw 会消耗你的扣子编程积分（包括对话、大模型调用、图片生成等）。  - OpenClaw 的 Token 消耗较高，你可以选择**省流版**以节省 Token 成本。 |

## 操作步骤

### 步骤一：一键部署 OpenClaw

扣子编程提供了一键部署 OpenClaw 的极简方案，在扣子编程首页单击**立刻领取**按钮，后台将自动完成项目的创建和部署。只需三分钟，即可获得一个 7×24 小时在线智能助手。

详细操作步骤如下：

1. 登录[扣子编程](https://code.coze.cn/)。

2. 在一键部署 OpenClaw 区域单击**立刻领取**。

-

  <img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/83d4b33d923d4d31b1e450683c034f7a~tplv-goo7wpa0wc-quality:q75.awebp" width="455">


3. 选择你的 OpenClaw 配置，并单击**确定**。

-

  |  |  |
  | --- | --- |
  | **配置** | **说明** |
  | 模型选择 | 为你的 OpenClaw 助理选择模型。默认为自动模式，你也可以选择豆包 2.0 等热门模型。部署后也可以随时切换模型。 |
  | 版本选择 | 扣子编程提供以下两种部署版本：  - **满血版**（默认）：OpenClaw 原版默认配置，无差异的性能体验。Token 消耗相对较多，适合需要极致性能的用户。  - **省流版**：在满血版的基础上调整了上下文压缩和心跳配置，Token 消耗相对较低，但部分场景下可能会有一些性能损失。省流版适合需要控制 Token 成本的用户。 |
  | 渠道配置 | 如果暂时无需配置飞书渠道，则跳过此步骤，直接单击**确定**。后续希望配置飞书渠道时，可参考[步骤二：配置飞书渠道](https://docs.coze.cn/tutorial/openclaw)。 |

### 步骤二：配置企业微信渠道

#### 1. 创建企业微信机器人

企业组织管理员和成员均可创建智能机器人

1. 登录[企业微信客户端](https://work.weixin.qq.com/#indexDownload)。

2. 在左侧目录树选择**工作台 > 智能办公**，单击**智能机器人**。

-

  <img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/2fe92392764941e2a0c8ec82bcf77739~tplv-goo7wpa0wc-quality:q75.awebp" width="442">


3. 单击创建机器人，并选择手动创建。

-

  <img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/a3b7c99dbca548bb953267819a755e6f~tplv-goo7wpa0wc-quality:q75.awebp" width="439">


4. 设置机器人的名称和头像。

-

  <img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/c16816ca249a417fb3d9b58beddfa66c~tplv-goo7wpa0wc-quality:q75.awebp" width="435">


#### 2. 开启长连接

通过长连接方式创建的智能机器人，支持被动回复多条消息、支持主动向用户发送消息。

1. 在**机器人配置**页面右下角，单击 **API模式创建**。

-

  <img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/db0f59296ca14595a335bce93e566181~tplv-goo7wpa0wc-quality:q75.awebp" width="431">


2. 在 **API 配置**区域，设置连接方式为**使用长连接**。

-

  <img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/2cec4fed21cf4e52b0ebd26a7860aa18~tplv-goo7wpa0wc-quality:q75.awebp" width="285">


3. 在**配置方法**区域，获取 Secret、复制 **Bot ID** 和 **Secret ID**，并单击**保存**。

- 至此，你已成功创建企业微信机器人。

- 在后续步骤中，需要输入 Bot ID 和 Secret ID 以将 OpenClaw 和此机器人绑定。

-

  <img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/44b12bf86bcc4b2ab09330ff7ca5e953~tplv-goo7wpa0wc-quality:q75.awebp" width="284">


#### 3. 配置企业微信插件

成功创建企业微信机器人，并获取 Bot ID 和 Secret 之后，你需要在扣子编程中配置企业微信插件。通过此步骤，你将为 OpenClaw 添加企业微信渠道，并和你的机器人绑定。

1. 回到[扣子编程](https://code.coze.cn/)，打开你的 OpenClaw 项目。

2. 在左下角对话框中输入以下内容，通过对话要求扣子 AI 配置企业微信插件。

-

  帮我设置企业微信渠道，我的 Bot ID 和 Secret 如下：

  - Bot ID：{替换为你的Bot ID}

  - Secret：{替换为你的 Secret}


<img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/51d5294978744e989863c1c116176b60~tplv-goo7wpa0wc-quality:q75.awebp" width="1230">

#### 4. 完成配对

配置完成后，首次在企业微信和 OpenClaw 个人助理对话时，你需要根据 OpenClaw 的提示完成配对。

1. 在企业微信和 OpenClaw 对话，并复制配对码。

- 在企业微信中搜索机器人名称，向其发送任意消息即可。

- > [!NOTE]
  >
  > **说明**
  >
  > 如果机器人未回复任何内容，原因是[3. 配置企业微信插件](https://docs.coze.cn/tutorial/openclaw_work_wechat)步骤配置失败，你可以尝试[通过终端配置企业微信](https://docs.coze.cn/tutorial/openclaw_work_wechat)，配置成功后再尝试配对。


-

  <img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/b17e93fedf2b4970926809f3c16dcf4d~tplv-goo7wpa0wc-quality:q75.awebp" width="353">


2. 打开扣子编程中的 OpenClaw 项目，在右侧预览页面输入配对码，要求完成配对。

- 你可以将步骤一中复制的文案直接发送给 OpenClaw。

3. OpenClaw 提示完成配对。

- 接下来你就可以在企业微信中和 OpenClaw 对话，体验效果。

-

  <img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/af64aa4283ce4c55a6ec684bcb05c454~tplv-goo7wpa0wc-quality:q75.awebp" width="338">

## 相关操作

### 通过终端配置企业微信

在执行[3. 配置企业微信插件](https://docs.coze.cn/tutorial/openclaw_work_wechat)时，由于模型的随机性，扣子 AI 有一定几率无法正确配置企业微信插件。你可以参考以下步骤，通过终端手动配置企业微信插件，将 OpenClaw 和你已创建的企业微信机器人绑定。

1. 在扣子编程的 OpenClaw 项目中，打开**终端**窗口。

-

  <img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/6c271acf712945d98ce9f055b3ee88a3~tplv-goo7wpa0wc-quality:q75.awebp" width="457">


2. 依次执行以下命令，配置企业微信插件。

-

  # 添加渠道

  openclaw channels add


3. 回显信息提示 Configure chat channels now?，回车表示选择 Yes。

-

  <img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/5312ed7496f74e9d8eb292c169c05066~tplv-goo7wpa0wc-quality:q75.awebp" width="464">


4. 回显信息提示 select a channel，选择 channel 为「企业微信」。

-

  <img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/1374c18a5a47432fb0c802143f9f906e~tplv-goo7wpa0wc-quality:q75.awebp" width="282">


5. 根据回显信息提示，输入企业微信机器人的 Bot ID 和 Secret ID。

-

  <img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/92695753a27f4ebf863f255fe08e71e4~tplv-goo7wpa0wc-quality:q75.awebp" width="297">


6. 移动光标，选择 Finished，结束配置。

-

  <img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/e4691ca719f941758323451e6b9093fc~tplv-goo7wpa0wc-quality:q75.awebp" width="291">


8. 页面提示设置 DM Policy（配对方式），默认为 pairing，选择 No 以跳过。

-

  <img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/3235270091e14c5190928f7aecc9b557~tplv-goo7wpa0wc-quality:q75.awebp" width="508">


11. 根据页面提示完成后续配置，完成时系统会提示 Channels updated.。

- 确认配置完成后，继续执行步骤 [4. 完成配对](https://docs.coze.cn/tutorial/openclaw_work_wechat)。
