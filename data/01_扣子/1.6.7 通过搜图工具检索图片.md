# 通过搜图工具检索图片

扣子提供了名为**搜索图片**的内置工具，在任务运行过程中自动调用，帮助你联网搜索查找在线图片，用于制作网页、画册、海报等。

## 应用场景

在扣子中，通过 AI Agent 执行任务时，往往需要联网搜索图片资源，为任务成果提供最新、更丰富、多样化的视觉内容。例如用户需要制作一个新闻网站，需要最新的产品图片、新闻事件的现场图片等，这些内容需要通过联网搜索来获取。

扣子内置了搜图工具，AI Agent 会根据任务目标，在需要在线图片资源时自行调用搜图工具，联网搜索图片资源，帮助用户制作网页、画册、海报等，辅助真实内容的呈现，丰富视觉效果。相较于 AI 生图，联网搜索图片会显著提高任务的运行效率。

## 使用方式

AI Agent 会根据任务目标，在需要在线图片资源时自行调用搜图工具，联网搜索图片资源。我们以制作少儿鸟类百科网页为例，演示扣子 AI Agent 调用搜索图片工具的具体过程。

1. 下发指令。

- 创建新任务时，在输入框内输入你的指令。例如“给4岁小男孩设计一篇英文教材，图文并茂的用适合这个年龄段孩子英文水平的词句来太阳系八大行星相关天文知识”。

-

  <img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/8042209c4722401a8bd6c3e4c57c8972~tplv-goo7wpa0wc-quality:q75.awebp" width="525">


2. 思考与规划。

- AI Agent 接收到提示词后，会立即启动意图识别机制，精准解析你的核心需求。随后 AI Agent 依据识别出的意图，自动构建详尽的任务规划，将复杂任务拆解为一系列可执行的子任务，并理清各个任务之间的依赖关系与先后顺序。

-

  <img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/726f6d74d8e1431c95e40e40c3bc935b~tplv-goo7wpa0wc-quality:q75.awebp" width="530">


3. 调用搜索图片工具联网搜索图片。

- AI Agent 会根据任务目标，在需要在线图片资源时自行调用搜图工具，联网搜索图片资源。例如你的要求是制作图文并茂的教材，AI Agent 会根据你的意图，调用搜图工具，联网搜索“太阳系八大行星”相关的图片资源。

4. 使用图片完成任务。

- 搜索到真实图片之后，AI Agent 会将检索到的这些图片资源添加到教程中展示。
