# 上架技能

你可以将扣子编程侧创建的技能上架到技能商店或企业市场，以便其他用户使用技能。

## 技能概述

扣子支持将技能上架到技能商店和企业市场，满足不同场景下的技能分发需求：

- **技能商店**：面向所有扣子用户的公开商店，技能可被扣子用户发现、安装和使用。你可将自主开发的技能设为付费模式上架，实现商业变现。这一机制既为开发者提供创意变现渠道，也为扣子用户提供丰富的技能选择。

- **企业市场**（仅企业版支持）：面向企业内部的专属技能分发中心，仅限本企业员工访问使用。适用于分发核心业务工具或内部专属技能，既能保护知识产权，又能提升内部协作效率。

## 使用限制

- 仅技能所有者能上架并管理技能。

- 不同技能类型的上架渠道及权限要求：

-

  |  |  |  |
  | --- | --- | --- |
  | **技能类型** | **技能商店** | **企业市场** |
  | 个人工作空间中的技能 | 技能所有者可以上架自己的技能。 | 不可上架。 |
  | 企业工作空间中的技能 | 企业超级管理员和管理员可以管控成员的技能上架权限，设置是否允许成员将企业工作空间中的技能上架到技能商店：  - **开启**：技能所有者可以上架。  - **关闭**：仅企业超级管理员和管理员可上架本人创建的技能。 | 技能所有者可以上架。 |


## 准备工作

在开始上架流程前，请确保你已完成以下准备：

- 已创建并部署技能，详细步骤可参考[制作技能](https://docs.coze.cn/cozespace/create_skill)。

- 如果上架付费技能，你需要先开通支付渠道，详细步骤可参考[开通收款账户](https://docs.coze.cn/guides/template_revenue_settlement)。

## 将技能上架到技能商店

### 步骤一：开启成员上架权限

> [!NOTE]
>
> **说明**
>
> 仅企业版需要执行该操作，个人版可跳过。

默认情况下，只有企业超级管理员和管理员能将企业工作空间中的技能上架到技能商店。管理员可以修改设置，为企业成员开启此权限。

1. 在[扣子编程](https://code.coze.cn/)左下角单击个人头像。

2. 找到要访问的企业，并单击**团队与企业管理**。

- 你也可以直接访问[扣子编程的企业管理页面](admin.coze.cn)，此域名默认打开最近一次访问过的企业管理页面。

-

  <img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/3af26d27e56f4b5f95d08c913f2309bf~tplv-goo7wpa0wc-quality:q75.awebp" width="218">


3. 在左侧导航栏中选择**企业成员管理**，在顶部单击**成员权限**页签。

4. 关闭**禁止企业成员上架 Skill 到扣子商店**的开关。

- **关闭**：企业成员可将企业工作空间中，本人创建的技能上架至技能商店。

- **开启**：仅企业超级管理员和管理员可将企业工作空间中，**本人创建**的技能上架至技能商店。

-

  <img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/ea614296ece044e08928d1d9ab537aa0~tplv-goo7wpa0wc-quality:q75.awebp" width="654">


### 步骤二：上架到技能商店

你可以将你创建的技能上架到技能商店，供扣子的其他用户发现并使用你的技能。

1. 在[扣子](https://space.coze.cn/)左侧导航栏选择**技能商店**。

2. 单击**我的技能**页签，在**我创建的**区域，选择目标技能。

3. 单击更多按钮，选择**上架到商店**。

-

  <img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/e5edeef96cc9461eb86478985513f589~tplv-goo7wpa0wc-quality:q75.awebp" width="362">


4. 在上架到商店页面，填写技能的介绍信息，配置说明如下。

-

  |  |  |
  | --- | --- |
  | **配置项**  | **说明**  |
  | **技能名称** | 为你的技能命名，支持中英文和数字，长度不超过 20 个字符。 |
  | **一句话描述** | 对技能的简明扼要的介绍，将展示在技能卡片中，帮助用户快速了解其核心功能，最多 200 字。 |
  | **封面图** | 技能在商店中展示的封面。你可以上传自己设计的图片，或使用 AI 生成功能快速创建。 |
  | **详细介绍** | 详细描述技能的功能、使用方法、应用场景和价值。你可以单击右上角的 **AI 生成**，让 AI 自动生成或优化技能的介绍文案。 |
  | **分类** | 为技能选择一个合适的分类，方便用户通过分类浏览时找到它。 |
  | **精选案例** | 提供 3 个在扣子中使用该技能的公开案例链接，以展示技能的实际应用效果。  1. 单击➕，在任务链接中粘贴扣子中使用该技能的案例链接，单击**识别链接**，系统将自动识别并填写任务名称和配图。  2. 单击**保存**。 |
  | **付费设置** | 设置技能的付费模式。  - **免费**：默认免费，所有用户均可直接使用。  - **按月付费**：必须提前开通收款账号，才可选择按月付费模式。 |
  | **是否开源** | 选择是否允许其他用户查看你的技能源码并进行复制改造。   - **开启**：其他用户可以对该技能进行复制改造，进行二次创作。  - **关闭**（默认）：不允许其他用户复制改造此技能。 |


5. 勾选**同意并签署技能上架开发者协议**。

6. 单击**上架到商店**。

-

  <img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/622ce55272c447bb9055e7a0ab4e96e3~tplv-goo7wpa0wc-quality:q75.awebp" width="485">


> [!NOTE]
>
> **说明**
>
> 提交上架申请后，扣子团队将对技能的图标、名称、描述、功能等进行审核，以确保其符合扣子商店规范。你可以在扣子站内信中查看上架审核的结果。

## 将技能上架到企业市场

企业市场是专为企业打造的内部技能分发中心。对于涉及核心业务逻辑或仅限内部使用的技能，上架到企业市场既能保护知识产权，又能提升内部协作效率。

> [!NOTE]
>
> **说明**
>
> **套餐限制**：仅企业版支持将技能上架到企业市场。

### 步骤一：管理员设置审核策略

默认情况下，成员无需审核即可将技能直接上架到企业市场。企业超级管理员或管理员可以修改此设置，增加上架审核环节。开启审核后，成员提交的技能必须先通过管理员审批，才能成功上架。

> [!NOTE]
>
> **说明**
>
> **角色限制**：企业超级管理员或管理员。

1. 在[扣子编程](https://code.coze.cn/)左下角单击个人头像。

2. 找到要访问的企业，并单击**团队与企业管理**。

- 你也可以直接访问[扣子编程的企业管理页面](admin.coze.cn)，此域名默认打开最近一次访问过的企业管理页面。

-

  <img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/3af26d27e56f4b5f95d08c913f2309bf~tplv-goo7wpa0wc-quality:q75.awebp" width="218">


3. 在左侧导航栏中选择**技能审核管理**。

4. 开启企业技能商店审核。

- **开启**：成员提交的技能需要由企业超级管理员或管理员审核通过后，才能在企业市场上架。

- **关闭**：（默认）成员提交的技能将直接上架企业市场，无需审批。

### 步骤三：管理员审核技能

如果开启了企业技能上架审核，员工提交的技能必须由企业超级管理员或管理员审批通过，才能在企业市场上架。审批通过后，该技能将对所有企业员工可见。

> [!NOTE]
>
> **说明**
>
> **角色限制**：企业超级管理员或管理员。

1. 在[扣子](https://space.coze.cn/)左侧导航栏选择**技能商店**。

2. 在右上角选择**企业管理** > **技能审核管理**。

3. 单击**待审核**标签筛选待审核的技能，在目标技能右侧，你可以查看技能详情、通过或拒绝上架。

## 管理我的技能

### 分享技能

上架技能商店后，你可以将技能分享给其他用户。

> [!NOTE]
>
> **说明**
>
> 技能所有者和普通用户均可分享技能。

1. 在[扣子](https://space.coze.cn/)左侧导航栏选择**技能商店**。

2. 在右上角选择**企业管理** > **技能管理**。

3. 单击目标技能，在技能详情页左上角单击分享按钮，可以将技能详情页分享给其他人。

- 对方打开链接后，可直接通过技能详情页购买或安装该技能。

### 更新上架

在扣子编程更新技能后，您可以将新版本上架至技能商店。更新完成后，所有已安装该技能的用户将自动升级至最新版本。

1. 在[扣子](https://space.coze.cn/)左侧导航栏选择**技能商店**。

2. 单击**我的技能**页签，在**我创建的**区域，选择目标技能。

3. 单击**更多 (⋮)** 按钮，选择**商店上架管理** > **更新上架技能**或**企业上架管理** > **更新上架技能**。

-

  <img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/6bd23917872b484fb21d7aee48b372d1~tplv-goo7wpa0wc-quality:q75.awebp" width="565">


### 下架技能

若技能商店或企业市场中的技能存在问题，或不希望被继续使用，你可以将其下架。

> [!NOTE]
>
> **说明**
>
> 技能下架后，用户将无法在技能商店中查找并安装该技能，具体影响如下：
>
> - **付费技能**：付费用户在有效期内仍可正常使用，直至其订阅到期，到期后无法再续费。
>
> - **免费技能**：已安装该技能的用户仍可正常使用。

**操作权限**

技能下架的操作权限要求：

- **技能商店**：仅所有者可下架。

- **企业市场**：技能所有者、企业超级管理员和管理员可下架。

**操作步骤**

所有者下架技能

管理员下架技能

1. 在[扣子](https://space.coze.cn/)左侧导航栏选择**技能商店**。

2. 单击**我的技能**页签，在**我创建的**区域，选择目标技能。

3. 单击**更多 (⋮)** 按钮，选择**商店上架管理** > **下架**或**企业上架管理** > **下架**。

<img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/f9cd9fd4b5cd4113a062f3b5e11ee53b~tplv-goo7wpa0wc-quality:q75.awebp" width="510">

### 移除技能

对于**已下架**的技能，如果后续不再使用，你可以将其从技能列表中移除。

> [!NOTE]
>
> **说明**
>
> 删除技能后，该技能将从**我的技能**列表中删除，扣子编程侧对应的技能并不会被删除。

1. 在[扣子](https://space.coze.cn/)左侧导航栏选择**技能商店**。

2. 单击**我的技能**页签，在**我创建的**区域，选择目标技能。

3. 单击**更多 (⋮)** 按钮，选择**删除**。
