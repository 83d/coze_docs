# 开始使用扣子

本文介绍扣子 AI Agent 的任务执行流程，包括输入提示词、任务分解、执行任务、交付结果和调整任务等关键步骤。

## 场景说明

扣子 AI Agent 是一款高效的生产力工具，具备智能拆解复杂任务、主动搜索、深度分析和精准执行任务的能力，能够处理复杂任务和琐碎事务，从而节省时间和精力。

例如你需要制作一份关于养老地产行业的研究报告时，将涉及大量的数据收集、分析和整理工作，通常需要耗费大量的时间和精力。而在扣子中，你只需输入提示词 做一个全面且深入的养老地产行业研究，AI Agent 便会自动解析你的需求，拆解任务并逐步执行，最终为你生成一份可视化的研究报告。

整个任务采用云端异步运行模式，AI Agent 会在后台自主完成任务，无需你全程参与。但在执行过程中，你可能需要与 AI Agent 进行交互，例如输入网址、用户名与密码等。

## 步骤一：输入提示词

输入任务提示词是与 AI Agent 交互的第一步，通过简洁明了的自然语言告诉 AI Agent，你想要完成的任务。具体流程如下：

1. 明确你的需求。

- 在输入提示词前，先明确你想要完成的任务。确保你的需求具体且明确，以便 AI Agent 能够准确理解并执行。例如：

- 如果你需要研究养老地产行业，可以输入 做一个全面且深入的养老地产行业研究。

- 如果你需要分析铜价走势，可以输入 帮我分析一下第二季度铜价的走势情况。

- 例如你需要筛选简历，可以输入我的公司正在寻求招聘一个 react 方向的前端工程师，要求能够独立负责一个中等规模网站的前端开发，熟悉 react 和 node.js，请帮我筛选出这份文件里面所有符合条件的简历，并且将他们的情况输出成一个可视化网页向我汇报。

2. 输入提示词。

- 在提示词输入框中，直接输入你的任务提示词。确保语言自然流畅，避免过于复杂或模糊的表述。

3. 扩展 AI Agent 能力。

- 如果需要拓展 AI Agent 能力，你可以在输入提示词时，为 AI Agent 添加技能。详细说明可参考[使用技能](https://docs.coze.cn/cozespace/using_skills)。

4. 选择开发模式。

- 根据需求，选择与 AI Agent 的开发模式。本文以探索模式为例，介绍扣子的使用步骤。

- 自动模式：让 AI Agent 自主决策使用探索模式还是规划模式。

- 探索模式：将任务完全交给 AI Agent，由其自主动态思考，完成速度更快。

- 规划模式：与 AI Agent 密切协同，让 AI Agent 协助规划步骤，分步帮你执行任务。

例如你想针对养老地产行业做一份研究报告时，可以输入 做一个全面且深入的养老地产行业研究。

## 步骤二：分解任务

AI Agent 接收到提示词后，会立即启动意图识别机制，精准解析你的核心需求。随后 AI Agent 依据识别出的意图，自动构建详尽的任务规划，将复杂任务拆解为一系列可执行的子任务，并理清各个任务之间的依赖关系与先后顺序。具体流程如下：

1. AI Agent 接收任务并启动意图识别。

- 当你向 AI Agent 输入提示词后，AI Agent 会立即启动意图识别机制，精准解析你的核心需求，理解任务的背景和目标。

2. AI Agent 构建任务规划。

- AI Agent 会根据识别出的意图，自动构建详尽的任务规划。AI Agent 会将复杂任务拆解为一系列可执行的子任务，并明确各个子任务之间的依赖关系与先后顺序。如果 AI Agent 对你的意图不够确定，它会主动向你提问，进一步明确你的需求。

3. 查看及确认子任务。

- 在任务对话框中，你可以看到 AI Agent 将任务分解为多个子步骤，并详细罗列每个子任务的具体内容。如果分解后的任务细节不符合你的预期，你可以单击**修改任务**，对任务内容进行修改，以确保任务按照你的期望推进。

4. 启动任务。

- 确认 AI Agent 罗列的计划符合你的需求后，你可以单击**开始任务**，启动任务。

例如针对养老地产行业研究， AI Agent 进行意图识别后，将把任务拆分为三个子步骤，并罗列具体的子任务内容，如下图所示：

## 步骤三：执行任务

> [!NOTE]
>
> **说明**
>
> 简单任务可能数分钟完成，复杂任务可能需要运行数小时。

单击**开始任务**后，AI Agent 开始逐步执行任务。具体执行流程如下：

1. AI Agent 自动执行任务。

- AI Agent 将根据已制定的任务计划，自动执行任务。

- **信息检索**：AI Agent 会自动调用浏览器检索各类信息。为了获取更丰富的信息，它还会模拟用户的网页操作行为，获取更多的网页内容和视觉信息，直到收集到的信息满足任务要求。任务对话框以及工作空间的**浏览器**页签中会展示 AI Agent 所访问的所有网页。你可以通过这些展示内容，实时了解 AI Agent 正在检索哪些信息。

- **数据处理：**AI Agent 会调用 websearch 工具模拟用户浏览网页的行为，抓取网页中的有效信息，并筛选出符合任务要求的内容。

- **数据保存**：AI Agent 会为每个子任务创建一个独立的文件夹，将所有收集到的信息保存到子任务文件中，为后续任务提供数据基础。

2. 用户监控任务。

- 任务的工作空间将全面展示任务执行过程中的各种记录，便于你监控与追溯任务。

- **实时跟随**：在工作空间的**实时跟随**页签中，你可以聚焦当前正在执行的动作，实时掌握任务的最新动态。

3. 用户调整任务。

- 如果在任务执行过程中发现任务的执行结果与你的预期存在偏差，你可以随时暂停任务并修改任务提示词，及时纠正。AI Agent 也具备自纠正能力，在执行任务过程中遇到无法解决的问题时，AI Agent 会自动中断任务，并立即向你发送对话消息，请求介入。如果在任务中断后的一小时内没有收到任何回复，AI Agent 将自动结束该任务。

4. AI Agent 整理归纳信息。

- 当所有任务完成后，AI Agent 会整理归纳执行过程中产生的所有内容。

例如 AI Agent 在处理养老地产行业研究任务时，会根据已制定的任务计划展示思考与搜索，并将相关内容保存到子任务文件夹中。

## 步骤四：交付结果

当所有的子任务标记完成后，AI Agent 将输出即用型结果，支持 PDF、网页等形式，并提供在线浏览及下载功能。结果内容支持以清晰的可视化形式呈现，包括动图、数据表格、交互式图表和仪表盘等。

例如，你可以单击 中国养老地产行业现状挑战与机遇分析.jsx 文件查看交付结果。

## 步骤五：调整任务

任务执行完成后，你可以查看并评估任务结果。如果发现某些部分不符合你的预期时，你可以输入更详细或更准确的任务描述，明确你的需求，继续推进任务。AI Agent 会重新为你生成任务结果。

> [!NOTE]
>
> **说明**
>
> 目前，由于资源有限，若在 2 小时内未进行回复，任务将自动结束，不支持继续调整任务。

## 相关操作

### 分享任务

扣子任务支持分享给其他用户，你可以在任务页面右上角，单击**分享**图标，分享任务。

- 仅限自己：只有你自己可以访问分享链接。

- 公开访问：所有获取到分享链接的用户都可以访问任务详情，便于团队协作或公开展示。

### 上传附件

扣子附件支持上传多种类型的文件。通过 AI Agent 智能处理文件内容，便于你高效完成数据分析、信息提取等任务，提升工作效率。你可以在输入提示词时，单击**附件**图标，上传附件。
