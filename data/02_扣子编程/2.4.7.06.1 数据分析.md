# 数据分析

借助分析看板，你能够全面跟踪和分析已发布的低代码智能体、应用和工作流的运行状况，包括 Token 消耗、积分消耗、用户数、对话数等基础运营数据、用户参与度、性能评估以及渠道运营洞察等，帮助你优化用户体验、提升运营效率、解决技术问题。

> [!NOTE]
>
> **说明**
>
> - 数据分析页面展示的 Token 消耗、积分消耗为预估值，实际扣减以火山账单为准。如何查看账单，请参考[查看账单](https://docs.coze.cn/coze_pro/bills_and_usage)。
>
> - 数据分析页面仅支持查看低代码智能体、应用和工作流的运行状况，暂不支持查看 AI 生成的智能体、应用和工作流的运行状况。

## 进入数据分析页面

1. 登录[扣子编程](https://code.coze.cn/home)。

2. 在左下角单击个人头像，选择企业，然后单击对应组织的**设置**图标。

3. 在**组织管理**页面的顶部选择**空间管理**，然后单击目标工作空间对应的**设置**。

3. 在**发布管理**页签下，单击对应的低代码智能体、应用或工作流，进入**分析**页面。

4. 在页面顶部选择时间段和发布渠道，查看分析数据。

## 使用限制

数据分析的使用限制如下表所示。

| **维度**  | **限制说明**  |
| --- | --- |
| 发布渠道  | 支持查看所有已发布的渠道的统计数据。 |
| 时间范围  | 默认展示过去7天的数据，最长可查看180天的统计数据。  |
| 数据更新时间  | 分析页面的数据将在次日更新，暂不支持查看当天的使用数据。  |
| 权限  | - 空间成员仅能查看自己作为所有者或协作者的应用、智能体和工作流的分析数据，添加智能体协作者的操作可参考[协同管理低代码智能体](https://docs.coze.cn/guides/collaborate_agent)。  - 空间所有者和管理员支持查看所有应用、智能体和工作流的分析数据。  |

## 数据分析图表说明

**分析**页面主要展示智能体、应用和工作流的基础运营数据、用户参与度、性能评估以及渠道运营数据。你可以根据这些数据了解过去一段时间内用户量的变化趋势、用户留存情况以及各发布渠道的运营数据等。

### 基本概念

本文中涉及的一些基本概念说明如下：

- **活跃用户**：指与智能体或应用至少进行过一轮有效对话的用户。

- **新用户**：指首次和智能体或应用进行对话的用户，此用户历史上从未和智能体或应用进行过对话。

- **对话**：用户与智能体或应用之间的一次有效交互，包含用户的询问、智能体或应用的有效回应。如果智能体或应用由于系统限流或其他技术因素未能提供有效回复，则不视为成功的对话。

- **对话次数**：用户与智能体或应用之间的有效对话次数。用户询问后，智能体或应用进行了回应，即计为一次有效对话。

- **对话时长：**在特定时间段内，特定用户与智能体或应用之间所有成功对话的累计时长。注意，对话时长不包括两次对话之间的间隔时间，只计算实际交互的持续时间。

- **会话**：用户围绕特定主题或任务与智能体或应用进行的一系列连续对话。如果新的对话开始时间与上一次对话结束时间之间的间隔超过 30 分钟，系统会将之后的对话视为新会话的开始。

### 累计指标

在**累计指标**区域，你可以查看该智能体、应用或工作流发布后的累计 Token 消耗、累计积分消耗、累计用户数和累计对话数。

>
> **说明**
>
> 该区域的图表不受时间段筛选的影响。

### 概览

在概览区域，你可以查看智能体、应用和工作流的用户数、新用户数、对话数等数据，以判断其整体运营情况。

<img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/fabceeb187bf482995bb97befdeadd27~tplv-goo7wpa0wc-quality:q75.awebp" width="626">

| **图表名称** | **说明** |
| --- | --- |
| 用户数 | 在指定时间段内，统计各发布渠道的以下数据，并展示与上个统计周期相比的数据变化趋势：  - 智能体：与智能体进行过对话的用户总数。  - 应用：运行应用的用户数。  - 工作流：工作流的运行数。 |
| 新用户数 | 在指定时间段内，统计各发布渠道的新用户数，并展示与上个统计周期相比的数据变化趋势。 |
| 对话数   | 在指定时间段内，统计各发布渠道的以下数据，并展示与上个统计周期相比的数据变化趋势。  - 智能体：与智能体的总对话次数。  - 应用：应用的调用次数。  - 工作流：工作流的运行次数。 |
| Token 消耗量 | 在指定时间段内，统计该智能体、应用或工作流在豆包、官方渠道、公共渠道等所有发布渠道累积的 Token 消耗量，并展示与上个统计周期相比的数据变化趋势。Token 消耗量反映了智能体、应用和工作流的资源使用情况。 |
| 积分消耗量 | 在指定时间段内，统计该智能体、应用或工作流在豆包、官方渠道、公共渠道等所有发布渠道累积的积分消耗量，并展示与上个统计周期相比的数据变化趋势。 |
| 用户来源 | 在指定时间段内，通过饼图展示智能体或应用的累计对话数在各发布渠道的分布情况，可直观查看各渠道的用户数量及占比。具体数据如下：  - 智能体：智能体的累计对话数量。  - 应用：应用的累计调用次数。  [!NOTE]  **说明**  仅智能体和应用支持查看用户来源，工作流不支持用户来源图表。  |

### 用户参与度

在用户参与度区域，你可以通过用户活跃度、用户留存数据了解智能体和应用的运营情况。

<img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/90aece8a43534fcb856ae9a014dffced~tplv-goo7wpa0wc-quality:q75.awebp" width="600">

| **指标名称** | **指标名称** | **说明** |
| --- | --- | --- |
| 用户活跃 | 对话数 | 在指定时段内、指定渠道下，所有用户与智能体或应用的总对话次数。 |
|  | 活跃用户数 | 在指定时段、指定渠道下，智能体或应用的活跃用户总数。 |
|  | 新用户数 | 在指定时段内、指定渠道下，首次与智能体或应用进行对话的用户总数。 |
|  | 人均对话数 | 在指定时段内、指定渠道下，人均和智能体对话次数。 |
|  | 新用户对话数 | 在指定时段内、指定渠道下，新用户与智能体或应用的对话次数。 |
|  | 新用户人均对话数 | 在指定时段内、指定渠道下，新用户人均与智能体或应用的对话次数。 |
| 用户留存 | 次日留存率 | 在统计周期内，第一天内新增的用户中，第二日仍然活跃的用户占比。 |
|  | 7 日留存率 | 在统计周期内，第一天的活跃用户中，第八日仍然活跃的用户占比。 |
|  | 月用户留存率 | 在统计周期内，第一天的活跃用户中，在下个月仍然活跃的用户占比。 |
|  | 次日新用户留存率 | 在统计周期内，首次与智能体或应用进行对话的新用户中，第二日再次对话的用户所占的比例。 |
|  | 7 日新用户留存率 | 在统计周期内，首次与智能体或应用进行对话的新用户中，第八日再次对话的用户所占的比例。 |
|  | 月新用户留存率 | 在统计周期内，首次与智能体或应用进行对话的新用户中，下个月再次对话的用户所占的比例。 |

### 性能

仅应用和工作流支持查看性能图表。在**性能**区域，你可以查看应用或工作流的性能指标，包括响应成功率、平均响应时长和错误码统计，从而优化应用和工作流的运行效率。

<img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/1b990570f5f74a0a9cd337e7c2eb3067~tplv-goo7wpa0wc-quality:q75.awebp" width="600">

| **图表名称** | **说明** |
| --- | --- |
| 响应成功率 | 展示应用或工作流在指定时间段内的响应成功率，并展示与上个统计周期相比的数据变化趋势。 |
| 平均响应时长 | 展示应用或工作流在指定时间段内的平均响应时长，并展示与上个统计周期相比的数据变化趋势。 |
| 错误码统计 | 展示应用或工作流在指定时间段内出现的所有错误码和错误信息，并展示与上个统计周期相比的数据变化趋势。单击右上角的**查看全部**，可以查看详细的错误码和错误信息。 |

### 渠道分析

仅智能体和应用支持查看渠道分析图表。

在**渠道分析**区域，你可以查看智能体和应用在各个发布渠道、指定时间段之内的运营数据，包括对话数、活跃用户数、新用户数、人均对话数、新用户对话数、新用户人均对话数、Token 消耗量、积分消耗量等。
