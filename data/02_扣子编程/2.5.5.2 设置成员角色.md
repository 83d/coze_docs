# 设置成员角色

添加企业成员后，企业超级管理员或管理员可以在成员列表中，修改企业成员的身份。

## 成员权限

企业和组织中，不同角色的具体权限说明如下，其中，✔️ 表示具备权限，- 表示没有权限。

  | **权限分类** | **说明** | **企业超级管理员** | **企业管理员** | **企业员工** | **访客** |
  | 套餐权限 | 享有企业版的套餐权益 | ✔️ | ✔️ | ✔️ | ✔️ |
  | 企业成员管理 | 成员增、删、修改成员身份 | ✔️ | ✔️ | - | - |
  |  | 查看成员列表 | ✔️ | ✔️ | ✔️ | ✔️ |
  | 企业设置 | 设置企业名称、图标等信息 | ✔️ | - | - | - |
  | 套餐管理 | 续订、退订套餐 | ✔️ | ✔️ | - | - |
  | 积分管理 | 购买、续订、退订积分等操作 | ✔️ | ✔️ | - | - |

- **组织级权限**

-

  | --- | --- | --- | --- | --- | --- |

2. 在左下角单击个人头像，选择企业> **团队与企业管理**。

## 设置成员在组织中的角色

组织创建者默认为组织的超级管理员，企业超级管理员申请加入组织后，直接加入成功且为组织的超级管理员。

**组织的超级管理员和管理员**可以修改内部成员的角色为组织的超级管理员、管理员或员工。组织角色的权限说明请参见[成员与权限](https://docs.coze.cn/guides/team_and_enterprise_overview)。

> [!NOTE]
>
> **说明**
>
> - 仅组织超级管理员能将员工设置为组织的超级管理员，组织管理员仅支持将员工设置为组织的管理员或员工。
>
> - 外部成员的角色只能是访客。
>
> - 组织超级管理员和管理员的数量暂时无限制。

## 常见问题

### 组织管理员是企业管理员吗？

不是。组织管理员和企业管理员是两个不同维度的角色，可能由同一人担任，也可能由不同人担任，具体取决于对应的超级管理员的授权。具体区别如下：

- **组织管理员**：由组织超级管理员授权设置，其角色归属 “组织” 维度。组织内的员工、企业管理员都可以担任组织管理员，主要负责组织成员、工作空间的管理。

- **企业管理员**：由企业超级管理员授权设置，其角色归属 “企业” 维度。主要负责企业成员、企业组织的管理。
