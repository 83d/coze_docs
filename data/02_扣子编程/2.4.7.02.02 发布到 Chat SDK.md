# 发布到 Chat SDK

编排低代码智能体并试运行通过后，可以将智能体发布到 Chat SDK，开发者可以在自己开发的网站中快速添加一个 AI 应用，为网站集成智能对话服务。

## 发布低代码智能体

1. 登录[扣子编程](https://code.coze.cn/home)。

2. 在页面顶部空间列表中选择目标工作空间。

3. 在**项目开发**页面，选择目标项目，进入低代码智能体的编排页面。

4. 在页面右上角，单击**发布**。

5. 在**发布**页面，选择 **Chat SDK**，并单击**发布****。**

- 未发布为 Chat SDK 的智能体，使用 Chat SDK 时会报错 提示：智能体已经被解绑。

6. 确认审核通过。

- 你可以在发布历史页面查看每次发布的审核结果，审核通过后再能执行后续操作安装 Chat SDK。查看发布历史的方式，可参考[管理低代码智能体版本](https://docs.coze.cn/guides/realese_history)。

## 后续操作

[安装并使用 Chat SDK](https://docs.coze.cn/developer_guides/install_web_sdk)

## 下架智能体

如果不再需要在该渠道中展示智能体，你可以选择将其下架。下架的操作步骤请参见[下架智能体](https://docs.coze.cn/guides/manage_published_project)。
