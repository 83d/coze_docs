# 发布为 API 服务

将低代码智能体发布为 API 服务后，可以通过调用 OpenAPI 的方式使用智能体程序。本文介绍如何将完成开发的智能体发布为 API 服务。

## 前提条件

要将低代码智能体发布为 API，必须满足以下条件：

- 在发布前已[创建个人令牌](https://www.coze.cn/open/oauth/pats)。详细说明可参考[鉴权方式概述](https://docs.coze.cn/developer_guides/authentication)。

## 发布智能体为 API

以下是将智能体发布为 API 的详细步骤：

1. 登录[扣子编程](https://code.coze.cn/home)。

2. 在页面顶部空间列表中选择目标工作空间。

3. 在**项目开发**页面，选择目标项目，进入低代码智能体的编排页面。

4. 在页面右上角，单击**发布**，进入智能体发布页面。

5. 在发布页面填写发布记录。

6. 在**选择发布平台 > API** 选项，选择 **API**。

7. 单击**发布**。

8. 发布成功后，单击完成。要查看发布历史，可以在智能体编排页面右上角，单击**发布历史**。

## 下架智能体

如果不再需要在该渠道中展示智能体，你可以选择将其下架。下架的操作步骤请参见[下架智能体](https://docs.coze.cn/guides/manage_published_project)。
