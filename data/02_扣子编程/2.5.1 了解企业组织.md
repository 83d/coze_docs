# 了解企业组织

扣子面向中小型团队和大型企业推出企业版订阅套餐（标准版、旗舰版）。企业版订阅套餐可享多种企业级专属权益，例如创建企业、组织、工作空间，使用 SSO 等企业级安全特性，实现中大型 AI 应用的多人协作。本章节介绍扣子编程中企业、组织和工作空间的概念，并描述企业与工作空间的关系。

## 企业

在扣子编程中，企业是一种虚拟架构，通常由购买企业版的扣子用户创建并管理。扣子用户购买企业版之后，即可创建企业、组织和工作空间，组织企业成员共同完成中大型 AI 应用的协同开发。企业组织下的数据与扣子用户的个人资源与数据互相隔离，企业成员需要切换到企业身份之下，才能使用企业资源与数据、在企业中搭建内部 AI 应用。

企业中的资源和套餐权益供内部成员共享，包括企业版中的积分、音色等。此外，所有成员均可体验与使用企业版的高级特性，例如音色管理、智能音视频能力等；成员搭建的 AI 应用也享有一系列企业级安全特性，保障企业的信息与数据安全。

## 组织

组织是企业下的虚拟分组，其中包含多个工作空间，便于集中管理不同工作空间下的成员、资源及权限。在大中型企业中，通常根据部门或项目维度将工作空间划分到不同组织，实现多业务部门的独立运作与统一管理。通过组织这一维度，企业可精细化管理成员与权限、实现资源的规范化配置，提高企业内部的协作效率。

每个企业包含一个默认组织，用户加入企业时将自动加入该默认组织。企业旗舰版支持创建多个组织，企业的超级管理员和管理员可以按部门或项目创建组织和工作空间、将员工分配到不同组织和工作空间中进行资源管理和协作。

例如，某大型科技企业购买了企业旗舰版，企业的超级管理员或管理员根据公司架构，按部门创建了研发部、产品部、运营部等多个组织，并指定各部门负责人担任对应组织的超级管理员或管理员。各组织的超级管理员和管理员将部门成员添加至所属组织，并在组织内创建工作空间，组织成员在工作空间内创建智能体、工作流等资源。

> [!NOTE]
>
> **说明**
>
> **企业标准版**仅提供一个默认组织；**企业旗舰版**和**原企业版**最多 20 个组织。

## 工作空间

企业中创建的技能、应用、智能体和工作流均保存在工作空间中，不同空间内的资源和数据权限相互隔离。工作空间中可以添加多个企业成员，成员在空间范围内实现多人协作。每个组织中可以创建多个工作空间，但每个工作空间只能隶属于一个组织。

## 成员与权限

> [!NOTE]
>
> **说明**
>
> - **企业标准版**可添加 2～ 100 个成员，包括超级管理员、管理员和访客。
>
> - **企业旗舰版**和**原企业版**无成员数量限制。

企业成员指已加入企业组织的扣子用户，他们产生的任何资源消耗都会被统计为企业的支出，并扣减企业的资源额度、积分，如果剩余积分不足，则换算成现金账单，由超级管理员统一支付。

此外，企业和组织中存在不同的角色和分工，各个角色的操作权限不同。企业中的角色包括以下几种：

| **维度** | **角色** | **说明** |
| --- | --- | --- |
| 企业 | 企业超级管理员 | 购买企业版时，系统会自动指定主账号对应的用户为超级管理员，超级管理员也可以手动添加超级管理员，拥有企业的管理权限。 |
|  | 企业管理员 | 由超级管理员授权的员工，协助管理企业，拥有成员管理、组织管理等部分权限。 |
|  | 员工 | 由企业超级管理员创建并添加到企业中的内部成员（子用户），仅对本人创建的资源享有管理权限，无法操作其他成员的资源。员工本质上属于企业超管的子账号，被分配给企业内部的员工使用。 |
|  | 访客 | 由超级管理员或管理员邀请加入的外部成员，作为访客加入企业后共享企业积分和权益。访客既非企业的员工，也与超级管理员不属于同一个主账号。访客不能成为组织超级管理员、组织管理员、空间管理员。 |
| 组织 | 组织超级管理员 | 组织的创建者，或由其他组织超级管理员授权的组织成员，拥有组织内的管理权限。 |
|  | 组织管理员 | 由组织超级管理员授权的组织成员，拥有组织成员、工作空间的管理权限。 |
|  | 组织成员 | 企业的员工和访客加入组织后成为组织成员。 |

不同角色的具体权限说明可参考[成员权限](https://docs.coze.cn/guides/set_member_role)、[空间成员角色权限](https://docs.coze.cn/guides/set_role)。
