# 添加文字插件

[添加文字插件](https://www.coze.cn/store/plugin/7439197585790844978)用于对图片添加文字，例如为图片添加水印。你可以通过定义文字的透明度、大小、字体、旋转角度、位置、边距、以及是否铺满等属性，灵活定制文字样式。

## 使用限制

扣子主账号内所有子账号共享**添加文字插件**的并发限制 ，其值为 10。

## 计费说明

添加文字插件根据插件调用次数计费，对应的计费项及单价请参考[插件费用](https://docs.coze.cn/coze_pro/plugin_fee)。

## 配置说明

添加文字插件包含 add\_text\_to\_image 工具。调用该工具时，你需要上传原始图片。如果需要定制文字样式，可以配置文字的透明度、大小、字体、旋转角度、位置、边距、以及是否铺满等参数。

### 输入参数

输入参数说明如下表所示：

| **界面参数** | **说明** |
| --- | --- |
| 底图 | 设置原始图片来源，必填参数。  支持上传图片或引用上游节点的输出参数。  [!NOTE]  **说明**  在调试节点效果时，建议上传一张示例图，用于调整参数配置，并仅测试该节点，快速查看效果。  |
| 文字内容 | 输入待添加的文字内容，必填参数。中文字符串的最大字节长度为 64 个字符。 |
| 文字颜色 | 设置文字的颜色，格式为十六进制颜色码。  - 取值范围： 000000 ～ FFFFFF  - 默认值：000000（黑色）。 |
| 平铺 | 设置是否将文字铺满整张图片。  - 1：铺满。  - 0（默认值）：不铺满。 |
| 水平边距 | 设置文字距离图片边缘的水平距离。只有当文字的初始位置为左上、左中、左下、右上、右中、右下时，该配置才生效。  - 取值范围：[0,4096]。  - 默认值：10。  - 单位：px。 |
| 初始位置 | 设置文字的初始位置。  - nw：左上。  - north：中上。  - ne：右上。  - west：左中。  - center：中部。  - east：右中。  - sw：左下。  - south：中下。  - se（默认值）：右下。 |
| 旋转 | 设置文字按顺时针方向旋转的角度。  - 取值范围：[0,360] 。  - 默认值：0，表示不旋转。 |
| 文字阴影 | 设置文字阴影的透明度。  - 取值范围：[0,100]。  - 默认值：0，表示完全透明，无阴影。 |
| 文字大小 | 设置文字大小。  - 取值范围：(0,1000]。  - 默认值：40。  - 单位：px。 |
| 文字透明度 | 设置文字的透明度。  - 取值范围：[0,100]。  - 默认值：100， 表示完全不透明。 |
| 文字字体 | 设置文字字体，支持设置为文泉驿正黑（默认）、文泉微米黑、方正书宋、方正楷体、方正黑体、方正仿宋。 |
| 垂直边距 | 设置文字的垂直边距，即文字距离图片边缘的垂直距离。 只有当文字的初始位置为左上、中上、右上、左下、中下、右下时，该配置才生效。  - 取值范围：[0,4096]。  - 默认值：10。  - 单位：px。 |
| 中线偏移 | 设置文字的中线垂直偏移量。只有当文字的初始位置为左中、中部、右中时，该配置生效。  - 取值范围：[-1000,1000]，其中正值表示向上偏移，负值表示向下偏移。  - 默认值：0，不偏移。  - 单位：px。 |

### 输出参数

| **参数** | **说明** |
| --- | --- |
| data | 添加文字后的图片 URL。URL 有效期为 30 天，请及时转存。 |
| msg | 执行插件时的状态描述或错误提示信息。 |
