# 视频生成插件

扣子编程官方提供两款视频生成插件，[Doubao-视频生成 Pro](https://www.coze.cn/store/plugin/7519125835639685155?from=plugin_card) 和 [Doubao-视频生成 Lite](https://www.coze.cn/store/plugin/7519094063849766927?from=plugin_card)。支持基于文本提示词生成视频，也支持基于文本提示词、视频首帧、尾帧图片或参考图共同生成视频。

## 插件说明

[Doubao-视频生成 Pro](https://www.coze.cn/store/plugin/7519125835639685155?from=plugin_card) 和 [Doubao-视频生成 Lite](https://www.coze.cn/store/plugin/7519094063849766927?from=plugin_card) 插件分别基于火山引擎 [Doubao Seedance 1.0 pro](https://www.volcengine.com/docs/82379/1587798) 和 [Doubao Seedance 1.0 lite](https://www.volcengine.com/docs/82379/1553576) 模型开发。 基于提示词或图片生成视频的技术已被广泛应用于短视频平台爆款内容量产、教育知识可视化动画等多元场景，帮助用户快速将文本或图像转化为生动视频内容。

扣子编程提供视频生成插件点，你可以在低代码工作流中使用该插件生成单个视频，也可以使用循环节点嵌套视频生成插件来批量生成视频，提高视频生成的效率。由视频生成插件生成的视频，帧率统一为 24帧/秒。

扣子编程提供 [Doubao-视频生成 Pro](https://www.coze.cn/store/plugin/7519125835639685155?from=plugin_card) 和 [Doubao-视频生成 Lite](https://www.coze.cn/store/plugin/7519094063849766927?from=plugin_card) 两款插件，其区别如下：

| **维度** | **Doubao-视频生成 Pro** | **Doubao-视频生成 Lite** | |  |
| --- | --- | --- | --- | --- |
| 模型 | doubao-seedance-pro | doubao-seedance-1-0-lite-i2v | doubao-seedance-1-0-lite-t2v |
| 支持的能力 | - 图生视频-首尾帧  - 图生视频-首帧  - 文生视频 | - 图生视频-参考图  - 图生视频-首尾帧  - 图生视频-首帧 | 文生视频 |
| 并发限制 | 10  | 5 | 5 |

## 使用限制

- 并发限制：扣子主账号内所有子账号共享视频生成插件的并发限制 ，具体数值请参见上表。

- 上传的图片须满足如下要求：

- 格式：JPEG（JPG）、PNG、WEBP、BMP、TIFF、GIF。

- 宽高比（宽/高）： 0.4~2.5。

- 边长：300～6000px，即短边像素需大于 300px，长边像素需小于 6000px。

- 大小：小于 10MB。

## 费用说明

在使用 Doubao 视频生成插件时，将根据消耗的模型 token 数计费。具体价格可参考[模型费用](https://docs.coze.cn/coze_pro/model_fee)，不同分辨率的视频费用预估可参考[计费参考](https://docs.coze.cn/guides/video_generation_node)。

- 使用 **Doubao-视频生成 Lite** 插件会产生 Doubao-Seedance-1.0-lite 计费项。

- 使用 **Doubao-视频生成 Pro** 插件会产生 Doubao-Seedance-1.0-pro 计费项。

## Doubao-视频生成 Pro

[Doubao-视频生成 Pro](https://www.coze.cn/store/plugin/7519125835639685155?from=plugin_card) 插件具备独特的多镜头叙事能力，在语义理解与指令遵循能力上取得突破，能生成运动流畅、细节丰富、风格多样且具备影视级美感的 1080P 高清视频。该插件适用于专业广告短片、超现实奇幻创作、细腻纪实生活记录等多元场景，可有效缩短高质量视频的制作周期并降低成本投入。

### 输入参数

| **参数名称** | **类型** | **是否必选** | **说明** |
| --- | --- | --- | --- |
| text | String | 必选 | 生成视频的提示词，用于下达视频生成相关的指令，也就是你对视频的描述。提示词越详细、越具体，生成的视频越符合你的期望。  提示词示例请参考 [Seedance-1.0-pro&pro-fast 提示词指南](https://www.volcengine.com/docs/82379/1631633)。 |
| camerafixed | Boolean | 可选 | 控制是否固定镜头，枚举值：  - false：根据提示词产生运镜。  - true：镜头固定。 |
| duration | Integer | 可选 | 生成视频的时长，单位为秒，取值范围： 3~12 秒。 |
| first\_frame\_url | String | 可选 | 指定生成视频首帧图片的 URL 地址。 |
| last\_frame\_url | String | 可选 | 指定生成视频尾帧图片的 URL 地址。 |
| model | String | 可选 | 指定用于生成视频的模型，默认值为 “doubao-seedance-pro”。 |
| ratio | String | 可选 | 生成视频的宽高比例，包括：21:9、16:9、4:3、1:1、3:4、9:16、9:21、keep\_ratio（生成视频的宽高比与上传图片宽高比保持一致）。 |
| resolution | String | 可选 | 生成视频的分辨率，例如 480p、720p、1080p。 |
| seed | Integer | 可选 | 用于控制生成内容的随机性，相同的种子值会生成类似的视频内容。  - -1（默认值）：系统将随机生成视频，每次生成结果不同。  - 任意正整数：系统将基于该种子值生成相似的视频，确保结果可复现。 |
| watermark | Boolean | 可选 | 控制生成的视频是否包含水印，枚举值：  - false：生成的视频不含水印。  - true：生成的视频含有 **AI生成**字样的水印。 |

### 输出参数

| **参数名称** | **类型** | **说明** |
| --- | --- | --- |
| code | Integer | 请求结果状态码，0 表示请求处理成功。 |
| data | Object | 视频生成的核心数据集合，包含视频信息、任务信息等。 |
| data.content.video\_url | String | 生成的视频的访问 URL，可直接用于播放或下载视频。  [!NOTE]  **说明**  生成的视频 URL 有效期为 24 小时，请及时转存。  |
| data.created\_at | Integer | 视频生成的 Unix 时间戳，单位为秒。 |
| data.updated\_at | Integer | 视频生成任务的最后更新时间戳，单位为秒，通常为任务完成时间。 |
| data.error.code | String | 错误编码，无错误时为空字符串。 |
| data.error.message | String | 错误描述信息，无错误时为空字符串。 |
| data.id | String | 视频生成任务的 ID，可用于查询任务状态。 |
| data.model | String | 生成视频的模型名称。 |
| data.status | String | 视频生成任务的状态，succeeded 表示任务已成功完成。 |
| data.usage.completion\_tokens | Integer | 模型输出视频花费的 token 数量。 |
| data.usage.total\_tokens | Integer | 本次请求消耗的总 token 数量。  视频生成模型不统计输入 token，输入 token 为 0，故 **total\_tokens**=**completion\_tokens**。 |
| log\_id | String | 请求的日志 ID，用于问题排查。 |
| msg | String | 插件执行状态，success 表示执行成功。 |

## Doubao-视频生成 Lite
