# 创建表格知识库

扣子知识库提供高效便捷的方式来存储和管理外部数据（包括文本、表格及图片），使低代码智能体可以与指定数据进行交互，提升回复内容的准确性和可用性。本文将介绍如何通过本地文档、API、飞书、自定义等渠道上传表格到知识库。

## 注意事项

创建知识库前，请先阅读[知识库概述](https://docs.coze.cn/guides/knowledge)、[使用限制](https://docs.coze.cn/guides/knowledge_limits)了解其功能特性及使用限制。

## 操作流程

参考以下操作，上传表格数据到知识库。

1. 登录[扣子编程](https://code.coze.cn/home)。

2. 在页面顶部选择目标工作空间，然后在左侧导航栏中单击**资源库**。

3. 在页面右上角，选择 **+资源** > **知识库**。

## 配置说明

扣子编程支持从本地文档、API、飞书、自定义等渠道上传表格到知识库。不同导入类型，对应的配置有所不同，详细说明如下：

> [!NOTE]
>
> **说明**
>
> 每个表格知识库仅支持定义一种表结构，即一个知识库只能添加一张表格。当你通过不同渠道向该表格知识库添加表格数据时，需确保所添加的表格结构与知识库中定义的表格结构完全一致。

### 本地文档

扣子编程支持用户将本地的 Excel 文档和 CSV 文档上传到知识库。特定的配置说明如下：

| **操作** | **说明** |
| --- | --- |
| 上传配置 | 选择**导入类型**为**本地文档**后，你可以在**上传**页面，单击上传或拖拽文档到上传区域。 |
| 表结构配置 | 上传本地文档后，你可以在**表结构配置**页面，指定数据范围、确认表结构及配置索引。详细说明如下：  1. 指定数据范围：通过选择**数据表**、**表头**、**数据起始行**指定数据范围。  2. 确认表结构：系统已默认获取了表头的列名，你可以自定义修改列名，或删除无需导入的列。  3. 配置索引：指定某一列作为查询匹配的语义字段。在响应用户查询时，系统会将用户查询内容与该列内容进行对比，并根据内容的相似度返回匹配结果。 |

### API

扣子编程支持用户将 API 返回数据中的表格内容上传到知识库。

> [!NOTE]
>
> **说明**
>
> 通过 API 方式上传到表格知识库时， API 返回值最大支持 64KB。

| **操作** | **说明** |
| --- | --- |
| 上传配置 | 选择**导入类型**为**API**后，你可以在**新增API**页面，设置 API 相关信息。  - 网址 URL：输入以 .json 结尾的 API URL，例如 https://xxxx.json。  - 在公网环境中通过 HTTPS 协议执行 Get 请求获取到的数据，均支持导入。  - 更新频率：选择是否自动更新指定页面的内容及自动更新的频率。 |
| 表结构配置 | 上传表格后，你可以在**表结构配置**页面，确认表结构及配置索引。详细说明如下：  1. 确认表结构：系统已默认获取了表头的列名，你可以自定义修改列名，或删除无需导入的列。  2. 配置索引：指定某一列作为查询匹配的语义字段。在响应用户查询时，系统会将用户查询内容与该列内容进行对比，并根据内容的相似度返回匹配结果。 |

### 飞书

扣子编程支持用户将授权过的飞书表格文档上传到知识库。

| **操作** | **说明** |
| --- | --- |
| 上传配置 | 选择**导入类型**为**飞书**后，你可以在**选择文件**页面，选择目标文档，上传到知识库。  首次导入某个飞书账号的文档时，需要先根据页面提示完成授权和安装。支持绑定多个飞书账号。具体操作，请参考[管理数据源权限](https://docs.coze.cn/guides/manage_data_source_permissions)。  [!NOTE]  **说明**  - 支持导入**我的空间**下所有者为本人的表格文档以及**知识库Wiki**中的表格文档，暂不支持导入共享空间下的表格文档。  - 创建飞书知识库时，由于飞书官方限制，在**知识库Wiki**页签下筛选文档时，如果筛选结果超过 200 条，只展示前 200 条。  |
| 表结构配置 | 上传飞书文档后，你可以在**表结构配置**页面，指定数据范围、确认表结构及配置索引。详细说明如下：  - 指定数据范围：通过选择**数据表**、**表头**、**数据起始行**指定数据范围。  - 确认表结构：系统已默认获取了表头的列名，你可以自定义修改列名，或删除某一列名。  - 配置索引：指定某个字段作为查询匹配的语义字段。在响应用户查询时，会将用户查询内容与该字段内容的内容进行比较，根据相似度进行匹配。 |

### 自定义

扣子编程支持用户添加自定义的表格数据到知识库。

| **操作** | **说明** |
| --- | --- |
| 上传配置 | 选择**导入类型**为**自定义**后，你需要手动定义数据表结构及配置索引。  - 定义表结构：设置列名、数据类型。  - 配置索引：指定某个字段作为查询匹配的语义字段。在响应用户查询时，会将用户查询内容与该字段内容的内容进行比较，根据相似度进行匹配。 |
| 添加数据 | 定义表结构后，你可以手动为表格添加列值，也可以通过本地文档或 API 渠道导入值。导入时，新导入的表格结构必须与现有表格结构一致。 |

## 相关操作

创建知识库后，你可以在智能体或工作流中使用知识库。同时，你还可以依据业务发展的实际需求，对知识库进行更新、删除、更新数据源权限等操作。相关操作说明如下：

- [使用知识库](https://docs.coze.cn/guides/use_knowledge)：在智能体或工作流中添加知识库，丰富 AI 应用的知识范围，提高模型回复内容的可靠性。

- [维护知识库](https://docs.coze.cn/guides/maintain_knowledge)：根据业务变化，你可以对知识库进行停用、启用、编辑、删除等操作。

- [管理数据源权限](https://docs.coze.cn/guides/manage_data_source_permissions)：从飞书渠道上传表格数据到知识库时，需要获取数据源侧的授权。完成授权后，你可以随时移除授权或添加其他账号的授权。
