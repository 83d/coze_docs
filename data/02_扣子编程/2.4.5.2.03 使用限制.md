# 使用限制

使用知识库时，应注意以下限制：

| **资源** | **限制说明** |
| --- | --- |
| 知识库数量 | - 单个个人用户或单个企业成员最多可创建 1000 个知识库。  - 单个低代码智能体最多可绑定 150 个知识库。  - 每个应用最多可绑定 150 个知识库。 |
| 知识库文件 | - 文本知识库  - 每个知识库下最多支持添加 300 个文件。  - 上传的每个文件大小不超过 100 MB。其中，txt 文件、纯文本 Markdown 文件最大上传大小为 5 MB。  - 图片知识库  - 每个知识库下最多支持添加 300 个文件。  - 上传的每个文件大小不超过 20 MB。  - 表格知识库  - 每个知识库下最多支持添加 300 个文件。  - 上传的每个文件大小不超过 20 MB。  - 表格知识库中每个表格最多 50 列，10000 行。 |
| 知识库分段 | - 每个文本类型知识库的分段总数最大为 10000 个。  - 每个表格类型知识库的分段总数最多为 10000 行。  - 单个知识库分段的最大长度为 5000 字符。 |
| 知识库容量 | - 个人免费版：知识库空间容量最大为 1 GB。   - 个人进阶版：知识库空间免费容量为 10 GB。  - 个人高阶版：知识库空间免费容量为 10 GB。  - 个人旗舰版：知识库空间免费容量为 10 GB。  - 企业标准版：知识库空间免费容量为 100 GB。  - 企业旗舰版：知识库空间免费容量为 2 TB。  [!NOTE]  **说明**  - 在企业版中，主账号及其所有企业成员共享知识库空间。  - 在付费套餐中，知识库空间的免费额度被用完时，系统支持自动扩容，超出部分按量计费。详细说明，请参考[知识库空间](https://www.volcengine.com/docs/84458/1494489)。  |
| 内容 | 知识库中如果添加了邮箱地址、电话、姓名等可能涉及个人隐私的信息，低代码智能体可能在某些渠道中使用受限，例如飞书渠道会屏蔽包含邮箱地址的低代码智能体消息。 |
| 上传知识库 | API 方式上传知识库时，表格类型 API 返回值最大支持 64KB。 |
| 火山知识库配额 | 火山知识库的文档数量、切片数量、CPU 配额、QPS 上限等配额数据，请参考[知识库配额说明](https://www.volcengine.com/docs/84313/1339026)。 |
