# 管理发布产物

通过扣子编程，你可以搭建各种基于大模型的 AI 项目，例如智能体、应用、工作流。扣子编程提供 AI 项目的发布管理能力，支持查看工作空间中已发布的智能体、应用或工作流，查看其消息日志、分析数据、触发器执行记录，也可以一键从商店中下架智能体、应用。

## 查看发布列表

编排并发布智能体、应用或工作流之后，你可以在工作空间的**发布管理**页面查看发布产物列表，列表中展示发布产物的创建人、已发布平台等信息。

1. 登录[扣子编程](https://code.coze.cn/home)。

2. 在左下角单击个人头像，选择企业，然后单击对应组织的**设置**图标。

-

  <img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/e1eb91083b2e44cd8ffccc9e79e677e3~tplv-goo7wpa0wc-quality:q75.awebp" width="311">


3. 在**组织管理**页面的顶部选择**空间管理**，然后单击目标工作空间对应的**设置**。

3. 在顶部选择**发布管理**页签，在工作空间的**发布管理**页面查看发布产物列表，列表中展示发布产物的创建人、已发布平台等信息。

## 查看消息日志

在智能体、应用或工作流中，每条消息回复都会生成一条消息日志，这个消息日志详细记录了其回复问题的编排步骤和过程。消息日志帮助你了解智能体、应用或工作流的推理过程，以便你进行针对性的分析和优化。

1. 登录[扣子编程](https://code.coze.cn/home)。

2. 在左下角单击个人头像，选择企业，然后单击对应组织的**设置**图标。

-

  <img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/e1eb91083b2e44cd8ffccc9e79e677e3~tplv-goo7wpa0wc-quality:q75.awebp" width="311">


3. 在**组织管理**页面的顶部选择**空间管理**，然后单击目标工作空间对应的**设置**。

3. 在顶部选择**发布管理**页签，选择智能体、应用或工作流，可查看对应发布产物的运行日志列表。页面右侧的**统计**区域展示筛选范围内运行日志的相关统计数据。关于运行日志和统计数据的详细说明，可参考[查看消息列表](https://docs.coze.cn/guides/queries)。

- 不同扣子套餐支持的日志相关权益如下表。

-

  |  |  |  |  |  |
  | --- | --- | --- | --- | --- |
  | **功能** | **个人免费版** | **个人进阶版**  **个人高阶版**  **个人旗舰版** | **企业标准版** | **企业旗舰版** |
  | 消息日志存储天数 | 3 天 | 7 天 | 90 天 | 180 天 |
  | 上报的消息数量 | 5 万条/月 | 10 万条/月 | 10 万条/月 | 200 万条/月 |
  | 消息导出 | 不支持 | 每个智能体单次最多可导出 5000 条消息记录。 | | |  |  |

## 查看数据分析

对于已发布的智能体、应用和工作流，你可通过分析看板跟踪和分析其运行情况，包括基础运营数据、用户参与度、性能评估以及渠道运营洞察等维度。

1. 登录[扣子编程](https://code.coze.cn/home)。

2. 在左下角单击个人头像，选择企业，然后单击对应组织的**设置**图标。

-

  <img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/e1eb91083b2e44cd8ffccc9e79e677e3~tplv-goo7wpa0wc-quality:q75.awebp" width="311">


3. 在**组织管理**页面的顶部选择**空间管理**，然后单击目标工作空间对应的**设置**。

4. 在顶部选择**发布管理**页签，选择智能体、应用或工作流，可查看对应发布产物的数据分析图表。关于数据分析图表的详细说明，可参考[数据分析](https://docs.coze.cn/guides/data)。

## 下架智能体或应用

### 下架智能体

支持的渠道：目前支持从扣子官方渠道及自定义渠道下架智能体，但不包括豆包渠道。

操作步骤：

1. 登录[扣子编程](https://code.coze.cn/home)。

2. 在左下角单击个人头像，选择企业，然后单击对应组织的**设置**图标。

-

  <img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/e1eb91083b2e44cd8ffccc9e79e677e3~tplv-goo7wpa0wc-quality:q75.awebp" width="311">


3. 在**组织管理**页面的顶部选择**空间管理**，然后单击目标工作空间对应的**设置**。

3. 在顶部选择**发布管理**页签，在目标智能体右侧单击**下架**，选择需要下架的渠道，单击**下架**，在确认对话框中单击**下架**。

- > [!NOTE]
  >
  > **说明**
  >
  > - 仅智能体所有者可以下架智能体。
  >
  > - 目前暂不支持通过该入口下架发布到豆包的智能体，若需在豆包中删除扣子智能体，可参考[在豆包中删除智能体](https://docs.coze.cn/guides/publish_to_doubao)。

### 下架应用

支持的渠道：目前仅支持从扣子商店及扣子空间扩展库下架应用。

操作步骤：

1. 登录[扣子编程](https://code.coze.cn/home)。

2. 在左下角单击个人头像，选择企业，然后单击对应组织的**设置**图标。

-

  <img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/e1eb91083b2e44cd8ffccc9e79e677e3~tplv-goo7wpa0wc-quality:q75.awebp" width="311">


3. 在**组织管理**页面的顶部选择**空间管理**，然后单击目标工作空间对应的**设置**。

3. 在顶部选择**发布管理**页签，单击应用名称，在右上角单击**管理**，选择**扣子商店**或**扣子空间扩展库**，单击**下架**，在确认对话框中单击**下架**。

- > [!NOTE]
  >
  > **说明**
  >
  > 仅应用所有者可以下架应用。
