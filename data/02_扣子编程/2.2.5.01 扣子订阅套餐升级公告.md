# 扣子订阅套餐升级公告

扣子空间和扣子开发平台现已全新升级为扣子和扣子编程产品，支持 AI 编程方式开发技能（Skill）、智能体、工作流、应用等。为向用户提供更加优质的产品和服务，扣子全面升级了[订阅套餐](https://www.coze.cn/premium)、功能权益和服务模式，限时免费的 AI 编程功能将同步正式计费。

## 升级时间

扣子全新的订阅套餐及配套服务于**2026年1 月 18 日**正式上线，建议尽快升级至新的订阅套餐，以享受最新的扣子功能与服务。旧版套餐续费截止日期为**2026年12月31日**，逾期无法再购买原套餐。

另外，以下产品功能自**2026年1 月 19 日起**正式开始计费：

- **扣子对话任务**：7 天内所有扣子老用户均可免费体验。

- **AI 编程**：即AI 编程方式开发技能（Skill）、智能体、工作流、应用。扣子付费套餐的老用户 7 天内可免费体验。

## 升级内容

1. **新增核心功能**。

- **扣子**：支持使用**技能**、去除产物水印、播客自定义音色等。

- **扣子编程**：支持 AI 编程开发**技能（Skill）**、智能体、工作流、应用。

2. **订阅套餐档位调整**。

- 调整订阅套餐档位与对应的高级权益范围。新增个人高阶版、个人旗舰版，原团队版调整为企业标准版，原企业版调整为企业旗舰版，最新的套餐档位及权益可参考[订阅套餐](https://www.coze.cn/premium)。升级前后的套餐映射关系如下：

-

  |  |  |
  | --- | --- |
  | **旧套餐** | **新套餐** |
  | 个人免费版 | 个人免费版 |
  | 个人进阶版 | 个人进阶版 |
  | - | 个人高阶版 |
  | - | 个人旗舰版 |
  | 团队版 | 企业标准版 |
  | 企业版 | 企业旗舰版 |


- 专业版将于2026年5月30日正式下线。

3. **积分体系升级**。

- 扣子资源点改名为积分。全新的个人版套餐支持多种方式获取积分：首次登录领积分、每日登录领积分、参与活动领取积分等。

- 企业版积分按成员席位 1:1 赠送（69 元，含6.9万积分/人/月）。

## 常见问题

### 我还能续费使用原套餐吗?

- 符合续费条件的原个人进阶版、团队版、企业版用户可按原价格及折扣续费旧套餐，续费截止日期为2026年12月31日。详细信息，请参考[老用户优惠](https://docs.coze.cn/guides/20260119_coze_premium_upgraded)。

- 若当前套餐到期时间晚于2026年12月31日，不可再续期，可继续使用至到期，期间享受新版功能权益。

### 原团队版升级套餐后，会支持原企业版的权益吗？

不支持。原团队版升级到新的企业标准版之后，仍旧只能创建一个组织、员工数量上限 100 人，这些基础的权益范围没有变化。

### 原企业版升级到新企业版时，如何支付成员费用？

升级原企业队版到新企业版时，系统会自动退订老套餐及增购的企业版席位数，并跳转至新套餐购买页面。同时，系统会根据账号内的成员数量，自动核算购买人数及对应成员费用，并与套餐基础权益费合并。

如果你希望减少成员费用，可以在升级前移除不再需要的成员。具体操作，请参考[移除企业成员](https://docs.coze.cn/guides/remove_enterprise_member)。

### 升级到新个人版后，为什么原资源包不见了？

从原付费套餐升级到新个人版后，积分减少且在火山引擎费用中心的[资源包](https://console.volcengine.com/finance/resource-package)页面找不到原资源包购买记录。这是因为新个人版的支付结算链路已从**火山引擎**迁移至**扣子**，因此升级到新个人版时，原付费套餐及增购的资源包均会被自动退订、退款。后续个人版的购买记录需要在扣子中查看。具体操作，请参考[订阅管理](https://docs.coze.cn/coze_pro/personal_plan)。

> [!NOTE]
>
> **说明**
>
> 2026 年 1 月 18 日起，已购买的资源包名称不变，新购的资源包更名为**积分**。

### 升级到新个人版后，为什么会收到欠费通知？

原付费套餐内还有积分，但升级到新个人版后，收到了欠费通知，这是因为新个人版的支付结算链路已从**火山引擎**迁移至**扣子**，因此升级到新个人版时，原付费套餐及增购的积分均会自动退订、退款。并且受火山引擎出账延迟影响，部分已产生的用量可能在退订后结算，因此导致了欠费。你可以根据个人需求选择是否充值火山引擎账号，一般不直接影响扣子个人版的使用。

### 如何判断我当前使用的是新版还是旧版套餐？

在[扣子编程](https://code.coze.cn/home)的**订阅管理**页面，如果有升级提示横幅，则表示你当前仍使用的是旧版订阅套餐。
