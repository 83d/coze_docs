# 开关

开关组件用于开启或关闭某个选项，常用于设置。

## 属性设置

开关组件支持丰富的属性设置，例如常用设置、位置设置、尺寸设置、样式设置、变换设置、状态设置和可见性设置。

### 常用设置

| **属性** | **说明** |
| --- | --- |
| 标签 | 配置是否显示开关组件的标签，标签可以为开关组件提供文字描述，使用户能够更直观地理解开关的状态或功能。 |
| 标签内容 | 配置开关组件的标签内容，支持配置静态内容和动态绑定数据，详情请参见[设置组件内容参数](https://docs.coze.cn/guides/set_content_parameters)。  仅打开了标签设置开关时，才支持此配置项。默认标签内容为**请设置**。 |
| 默认值 | 配置开关组件默认值为开或关。 |
| 组件布局 | 配置标签与开关按钮的布局方式，支持上下布局和左右布局。 |
| 标签宽度 | 以百分比方式配置开关组件的标签的宽度。  仅组件布局方式为左右布局时，支持此配置项。 |
| 垂直对齐 | 配置开关组件的垂直对齐方式。  - 范围：顶部对齐、垂直居中对齐、底部对齐。  - 默认值：垂直居中对齐。 |
| 禁用态 | 可控制开关组件在什么情况下展示禁用状态。  - 表达式的结果为 true：开关组件呈禁用态。  - 表达式的结果为 false：开关组件呈可用态。  例如，通过一个开关组件来控制此组件是否禁用，可以将表达式设置为{{Switch1.value}}，当打开开关组件时，{{Switch1.value}}为true，此组件呈禁用态；当关闭开关组件时，{{Switch1.value}}为false，此组件呈可用态。 |
| 加载态 | 可控制开关组件在什么情况下展示加载状态。  - 表达式的结果为 true：开关组件呈加载态。  - 表达式的结果为 false：开关组件呈可用态。  例如，希望工作流在执行过程中，此组件呈加载状态，则可以在输入框中输入{{Workflow1.loading}}。注意，loading是数据源的一个状态，数据源的loading在请求过程中为true，在请求未开始或结束时为false。 |

### 位置设置

开关组件提供三种位置类型选项，以灵活配置组件在界面中的位置。

- 相对定位：堆叠布局下，元素自动向下或向右排列。

- 绝对定位：页面滚动时，绝对定位的元素会随页面一起滚动。

- 固定定位：页面滚动时，固定定位的元素会保持固定位置不变。

### 尺寸设置

| **属性** | **说明** |
| --- | --- |
| 宽度 | 配置开关组件的宽度值，支持以下宽度设置方式：  - 固定：固定的像素值，不会随容器大小变化。  - 百分比：元素的宽度占容器宽度的百分比。  - 填充容器：元素的宽度填满容器的可用空间。  - 适应内容：元素的宽度会随着内容的大小自动调整。 |
| 高度 | 配置开关组件的高度值，支持以下高度设置方式：  - 固定：固定的像素值，不会随容器大小变化。  - 百分比：元素的高度占容器高度的百分比。  - 填充容器：元素的高度填满容器的可用空间。  - 适应内容：元素的高度会随着内容的大小自动调整。 |
| 尺寸限制 | 配置开关组件的尺寸限制，支持配置最大高度、最小高度、最大宽度、最小宽度。  合理设置尺寸限制参数可以确保组件在不同屏幕尺寸下保持布局的稳定性和一致性，避免因屏幕尺寸差异而导致的布局问题。 |

### 样式设置

| **属性** | **说明** |
| --- | --- |
| 填充 | 配置开关组件背景的显示颜色。 |
| 圆角 | 配置开关组件背景的圆角，支持单独设置四个角的圆角。 |
| 内边距 | 配置开关组件背景的内边距，可以单独设置四个方向的内边距。 |
| 外边距 | 配置开关组件背景的外边距，可以单独设置四个方向的外边距。 |
| 边框 | 配置开关组件背景的边框样式。支持如下设置方式：  - 预设样式：提供预设的边框样式，可直接使用。  - 自定义样式：通过边框线的颜色、粗细和样式，自定义边框样式。 |
| 阴影 | 配置开关组件背景的阴影样式，增强组件的立体感。支持如下设置方式：  - 预设样式：提供预设的阴影样式，可直接使用。  - 自定义样式：支持通过样式、X偏移、Y偏移、模糊半径、扩展半径、颜色，自定义阴影样式。 |
| 模糊 | 配置开关组件的模糊效果，数值越大，模糊效果越明显。支持设置图层模糊和背景模糊。  - 图层模糊：当前图层内包含的所有元素都将被模糊处理。  - 背景模糊：支持模糊处理当前组件底部所覆盖部分。 |

### 变换设置

丰富界面视觉效果的参数。

| **属性** | **说明** |
| --- | --- |
| 缩放 | 配置开关组件大小的缩放比例。 |
| 旋转 | 配置开关组件围绕中心点旋转的角度。 |
| 位移 | 配置开关组件在水平和垂直方向上的移动距离。 |
| 倾斜 | 配置开关组件在水平和垂直方向上的倾斜角度。 |

### 状态设置

控制开关组件悬浮态、点击态和出现态的视觉效果参数。

| **属性** | **说明** |
| --- | --- |
| 悬浮效果 | 配置开关组件的悬浮态效果。具体配置说明如下：  - 样式效果：当鼠标悬浮到开关组件上时，开关组件的样式变化情况。包括背景色、边框等。详细说明，请参考[样式设置](https://docs.coze.cn/guides/switch_web)。  - 变换效果：当鼠标悬浮到开关组件上时，开关组件的动态变化情况。包括缩放、旋转等。详细说明，请参考[变换设置](https://docs.coze.cn/guides/switch_web)。  - 过渡效果：设置开关组件悬浮态的动画效果。  - 动画效果：支持设置为缓入缓出、缓入、缓出、线性等动画。  - 持续时间：设置动画的持续时间，单位为毫秒。  - 延迟时间：设置动画延迟出现的时间，单位为毫秒。  - 生效范围：设置动画的生效范围。  - 应用于全部变化：动画效果将应用于样式设置和变换设置所产生的所有变化。  - 仅应用于变换效果：动画效果仅应用于变换设置所产生的变化。 |
| 点击效果 | 配置开关组件的点击态效果。具体配置说明如下：  - 样式效果：点击开关组件时，开关组件的样式变化情况。详细说明，请参考[样式设置](https://docs.coze.cn/guides/switch_web)。  - 变换效果：点击开关组件时，开关组件的动态变化情况。详细说明，请参考[变换设置](https://docs.coze.cn/guides/switch_web)。  - 过渡效果：设置开关组件点击态的动画效果。  - 动画效果：支持设置为缓入缓出、缓入、缓出、线性等动画。  - 持续时间：设置动画的持续时间，单位为毫秒。  - 延迟时间：设置动画延迟出现的时间，单位为毫秒。  - 生效范围：设置动画的生效范围。  - 应用于全部变化：动画效果将应用于样式设置和变换设置所产生的所有变化。  - 仅应用于变换效果：动画效果仅应用于变换设置所产生的变化。 |
| 出场效果 | 设置开关组件在界面中出现时的动画效果。目前支持设置为淡入、缩放、模糊、水平翻转、垂直翻转、从下滑入、从上滑入、从左滑入、从右滑入。 |

### 可见性设置

控制开关组件隐藏或显示的参数，支持在特定条件下隐藏开关组件。

- 设置为 false：显示组件。

- 设置为 true：隐藏组件。

- 设置为变量：通过变量值（true 或 false）动态控制开关组件的可见性。

- 例如可以通过文本输入组件控制开关组件的可见性。文本输入组件 ID 为 Textarea1，当你将开关组件的**可见性**设置为{{ Textarea1.value}}后，在文本输入组件中输入文本时，{{ Textarea1.value}}值为 true，开关组件将被隐藏。

## 事件设置

通过配置开关组件的事件，可以为开关组件添加丰富的交互功能，以增强用户体验和界面的互动性。

| **事件项** | **说明** |
| --- | --- |
| 事件类型 | 支持当开关组件的状态发生变化时，触发预定义的事件。 |
| 组件方法 | 支持以下方法：  - 清除：清除组件的内容，是开关组件的状态恢复到默认状态。  - 清除验证：移除组件上的所有验证错误信息。  - 重置：将组件恢复到初始状态。  - 滚动到视图：将组件滚动到可视区域内。  - 设置数据：为组件设置或更新内容。  - 设置禁用：使组件变为禁用状态，用户无法与之交互。  - 设置隐藏：隐藏组件，使其不可见。 |
