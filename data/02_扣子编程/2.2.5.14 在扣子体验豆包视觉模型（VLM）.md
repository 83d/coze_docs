# 在扣子体验豆包视觉模型（VLM）

当你传入图片时，[视觉语言模型](https://www.volcengine.com/docs/82379/1362931)（VLM）可以理解图片里的视觉信息，并结合这些信息完成图片相关的任务，例如描述图片等。豆包模型家族中的 Doubao-vision-pro-32k 就是 VLM 的一种。基于RTC + 视觉大模型/语言大模型的强大理解能力，扣子支持和自己搭建的智能体进行低延迟高质量的视频实时通话，让智能体耳聪目明。

## 体验豆包视觉模型

1. 前往火山方舟控制台，开通 Doubao-vision-pro-32k 大模型。

- 操作步骤可参考[接入火山方舟模型](https://docs.coze.cn/guides/ark_model)。

2. 创建智能体，并且在**模型设置**中选择豆包**视觉模型**。

-

  <img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/4d214fa99224489d8ca9ca0eb524f1a7~tplv-goo7wpa0wc-quality:q75.awebp" width="233">


3. 将智能体发布为 API 服务。

-

  <img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/7f34e61be27c4779b57d0eef1e1c6883~tplv-goo7wpa0wc-quality:q75.awebp" width="542">


4. 访问网址 <https://www.coze.cn/open-platform/realtime/playground> 即可体验。
