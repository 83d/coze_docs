# 视频生成节点

低代码工作流中的视频生成节点，采用即梦同款视频模型，支持基于文本提示词生成视频，也支持基于文本提示词和视频首帧、尾帧图片共同生成视频。

## 节点说明

在 AI 智能处理场景下，基于提示词或图片生成视频的技术已被广泛应用到多元领域，例如短视频平台爆款内容量产、教育知识可视化动画等。扣子编程提供视频生成节点，你可以在低代码工作流中使用视频生成节点生成单个视频，也可以使用循环节点嵌套视频生成节点来批量生成视频，提高视频生成的效率和效果。由视频生成节点生成的视频，帧率统一为 24帧/秒。

- **模型选择**

- **扣子官方模型**：视频生成节点默认内置了 doubao-seedance-lite 模型、doubao-seedance-pro 模型和 doubao-seedance-1.5-pro 模型。所有扣子套餐用户均可使用。

- **方舟接入点**：如果是企业版（标准版、旗舰版）用户，支持开通并使用火山方舟的 doubao-seedance-1.0-lite-t2v、doubao-seedance-1.0-lite-i2v、doubao-seedance-1.0-pro 或 Doubao-Seedance-1.0-pro-fast 模型。具体的模型说明，请参考[视频生成](https://www.volcengine.com/docs/82379/1330310#视频生成能力)。

- > [!NOTE]
  >
  > **说明**
  >
  > 开通的火山方舟视频生成模型，只支持在**视频生成节点**中使用。

- **场景选择**

- 视频生成节点支持文生视频和图文生视频两种场景。两种场景均支持指定视频时长、分辨率、视频比例等配置，满足多样化的创作需求。

- **文生视频**：基于文本提示词生成视频。

- **图文生视频**：根据所选的模型，需要完成不同的配置。

- 使用 lite 类型的模型时，基于首帧图片、尾帧图片和文本提示词生成视频。

- 使用 pro 类型的模型时，基于首帧图片和文本提示词生成视频。

## 费用说明

使用视频生成节点将根据消耗的模型 token 数计费。具体价格可参考[模型费用](https://docs.coze.cn/coze_pro/model_fee)，不同分辨率的视频费用预估可参考[计费参考](https://docs.coze.cn/guides/video_generation_node)。

- doubao-seedance-lite 模型对应**视频生成 Lite** 计费项。

- doubao-seedance-pro 模型对应**视频生成 Pro** 计费项。

- doubao-seedance-1.5-pro 模型对应**视频生成 1.5Pro-有声**或**视频生成 1.5Pro-无声**计费项。

- 使用自开通的方舟模型时，由方舟模型服务收取费用。

## 前提条件

如果要使用自开通的方舟模型，则需要先完成如下操作。

- 接入方舟模型到扣子编程。具体操作，请参考[接入火山方舟模型](https://docs.coze.cn/guides/ark_model)。

- 其中，在接入模型时，需选择 doubao-seedance-1.0-lite-t2v、doubao-seedance-1.0-lite-i2v、doubao-seedance-1.0-pro 或 Doubao-Seedance-1.0-pro-fast 模型。

## 配置节点

### 模型设置

选择用于生成视频的模型，并设置生成视频的比例、分辨率、时长等信息。

扣子官方模型

方舟视频生成模型

选择**视频生成模型**为 doubao-seedance-lite、doubao-seedance-pro 或 doubao-seedance-1.5-pro 模型时，需要配置如下参数。不同模型对应的参数略有不同，以实际界面为准。

- **比例**：视频的宽高比例。

- **分辨率**：视频的分辨率。

- **时长**：视频的时长，单位：秒。

- **doubao-seedance-lite 模型**：5 秒或 10 秒。

- **doubao-seedance-pro 模型**：3～12 秒。

- **doubao-seedance-1.5-pro 模型**：4～12 秒。

- **固定镜头**：开启后镜头固定。如果未开启，则根据提示词产生运镜。

- **种子值**：用于控制生成内容的随机性，相同的种子值会生成类似的视频内容。

- -1（默认值）：系统将随机生成视频，每次生成结果不同。

- 任意正整数：系统将基于该种子值生成相似的视频，确保结果可复现。

- **是否包含水印**：打开开关后，将为视频添加 **AI生成**字样的水印。

- **是否包含音频**：doubao-seedance-1.5-pro 模型支持为生成的视频添加同步音频。

### 参考图片

在图文生成视频的场景中，需要提供参考图片，视频生成节点会基于参考图片生成完整的视频。

- **视频首帧**：视频首帧图片，image 类型，你可以直接上传一张图片，也可以引用上游节点的输出（image 类型的图片或 string 类型的图片 URL）。

- **视频尾帧**：视频尾帧图片，image 类型，你可以直接上传一张图片，也可以引用上游节点的输出（image 类型的图片或 string 类型的图片 URL）。

- 使用 lite 类型的模型时，必须设置尾帧图片。

- **参考图**：部分模型支持上传参考图，为模型提供具体的视觉参考。最多可上传 4 张图片。

上传的图片须满足如下要求：

- 格式：JPEG（JPG）、PNG、WEBP、BMP、TIFF、GIF。

- 宽高比（宽/高）： 0.4~2.5。

- 边长：300～6000px，即短边像素需大于 300px，长边像素需小于 6000px。

- 大小：小于 10MB。

### 输入

你可以添加输入参数，并在提示词中使用该输入参数，用于动态传入内容。输入参数可以指定为一个固定值，也可以引用上游节点的输出。

### 提示词

视频生成模型的提示词，用于下达视频生成相关的指令，也就是你对视频的描述。提示词越详细、越具体，生成的视频越符合你的期望。

提示词支持引用**输入**中定义的变量，引用方式为{{变量名}}。例如添加一个输入参数 place 并引用开始节点的 input 参数，然后在提示词中使用 {{place}}，当你通过输入节点传入草原、海边、操作等地点信息后，视频生成模型会根据传入的地点动态替换变量值，生成对应地点的视频。

### 输出

节点的输出参数固定为以下参数：

- video：模型生成的视频。通常是一个公开可访问的 URL 链接。

- > [!NOTE]
  >
  > **说明**
  >
  > 生成的视频 URL 有效期为 24 小时，请及时转存。


- msg：节点执行状态，success 表示执行成功。

## 计费参考

在调用视频生成节点，系统将根据画面比例、分辨率及视频时长计算每个视频的 token 用量，然后根据 Token 用量计算每个生成视频的费用。其中，估算 token 用量的公式为(宽 × 高 × 帧率 × 时长)/1024，帧率固定为 24 帧/秒。

## 常见问题

### 如何提高视频生成效果？

在使用视频生成节点时，可参考如下建议配置节点，提高视频生成效果。

- 用简洁准确的自然语言写出你想要的效果。

- 如果有较为明确的效果预期，建议先用生图模型生成符合预期的图片，再用图生视频进行视频片段的生成。

- 图生视频时请尽量上传高清高质量的图片，上传图片的质量对图生视频影响较大。

- 当生成的视频不符合预期时，建议修改提示词，将抽象描述换成具象描述，并注意删除不重要的部分，将重要内容前置。
