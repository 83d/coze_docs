# 组件概述

搭建小程序端用户界面，支持使用多种类型组件，例如输入组件、展示组件和布局组件，每一类组件包含多个不同的组件。

| **组件分类** | **组件名称** | **说明** |
| --- | --- | --- |
| 布局组件 | [横向列表](https://docs.coze.cn/guides/horizontal_list_mini_program) | 用于横向展示数据项的列表视图。 |
|  | [纵向列表](https://docs.coze.cn/guides/vertical_list_mini_program) | 用于纵向展示数据项的列表视图。 |
|  | [宫格列表](https://docs.coze.cn/guides/grid_list_mini_program) | 用于以宫格形式展示数据项的列表视图。 |
|  | [瀑布流列表](https://docs.coze.cn/guides/waterfall_list_mini_program) | 用于以瀑布流形式展示数据项的列表视图。 |
|  | [容器](https://docs.coze.cn/guides/container_mini_program) | 用于包含和对齐一组子组件，是构建复杂布局的基础。 |
| 展示组件 | [文本](https://docs.coze.cn/guides/text_mini_program) | 用于显示静态文本内容。 |
|  | [图标](https://docs.coze.cn/guides/icon_mini_program) | 用于展示图标，增强界面的视觉效果和交互性。 |
|  | [Markdown](https://docs.coze.cn/guides/markdown_mini_program) | 用于渲染 Markdown 格式的文本。 |
|  | [音频](https://docs.coze.cn/guides/audio) | 用于播放音频内容。 |
|  | [图片](https://docs.coze.cn/guides/image_mini_program) | 用于展示图片。 |
|  | [Lottie 动画](https://docs.coze.cn/guides/lottie_mini_program) | 用于展示 Lottie 动画。 |
|  | [轮播](https://docs.coze.cn/guides/carousel_mini_program) | 用于在有限空间内循环展示一系列图片。 |
| 输入组件 | [按钮](https://docs.coze.cn/guides/button_mini_program) | 用于触发事件或动作，例如提交表单。 |
|  | [表单](https://docs.coze.cn/guides/form_mini_program) | 用于收集用户输入的数据，通常与多个输入字段一起使用。 |
|  | [单行输入](https://docs.coze.cn/guides/single_line_input) | 单行输入组件用于接收用户的单行文本输入，适用于需要简短回答的场景，例如搜索框、密码输入、电话号码等。 |
|  | [多行输入](https://docs.coze.cn/guides/multi_line_input) | 用于接收用户的多行文本输入，适用于需要用户输入较多文本的场景，例如评论、文章撰写、长描述等。 |
|  | [下拉选择](https://docs.coze.cn/guides/drop-down_selection) | 用于从预定义的选项列表中选择一个选项。 |
|  | [数字输入](https://docs.coze.cn/guides/numeric_Input_mini_program) | 用于输入数字，适用于需要精确数值的场景。 |
|  | [图片上传](https://docs.coze.cn/guides/image_upload_mini_program) | 用于上传和展示图片，常用于图像内容的管理。 |
|  | [单选按钮](https://docs.coze.cn/guides/radio_button_mini_program) | 用于在多个选项中选择一个，确保用户只能选择一个选项。 |
| 导航组件 | [标签栏](https://docs.coze.cn/guides/tab_bar_mini_program) | 用于在不同的内容区域之间进行切换，每个标签代表一个不同的内容视图。 |
| AI 组件 | [AI 对话](https://docs.coze.cn/guides/ai_chat_mini_program) | 用于搭建对话式 AI 应用，支持绑定对话流、设置开场白和快捷指令。 |
