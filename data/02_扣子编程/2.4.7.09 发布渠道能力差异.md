# 发布渠道能力差异

本文介绍扣子发布渠道支持的工作流节点的差异和能力差异，在发布智能体和应用时，请根据实际需求选择合适的发布渠道。

## 工作流节点

多数工作流节点在各发布渠道均支持，仅个别节点在不同发布渠道的支持情况有差异，具体如下表所示。

✔️表示支持，- 表示不支持。

| **节点** | |  | **豆包** | **Chat SDK** | **Web SDK** | **API** | **微信/抖音小程序** | **通讯或社交平台**  **微信、飞书、多维表格、掘金** |
| --- | --- | --- | --- | --- | --- | --- | --- | --- |
| 问答节点 | 应用 | - | ✔️**＊**① | ✔️ | ✔️ | ✔️ | ✔️＊③ |
|  | 智能体 | - | ✔️**＊**② | - | ✔️ | ✔️**＊**② | ✔️＊② |
| 输入节点 | 应用 | - | ✔️ | ✔️ | ✔️ | ✔️ | - |
|  | 智能体 | - | - | - | ✔️ | - | - |
| 消息节点（例如创建消息节点） | 智能体/应用 | - | ✔️ | ✔️ | ✔️ | ✔️ | - |
| 会话历史节点 | 智能体/应用 | - | ✔️ | ✔️ | ✔️ | ✔️ | ✔️ |

> [!NOTE]
>
> **说明**
>
> - **＊**①：应用发布 Chat SDK 时，仅对话流搭建的应用支持问答节点，工作流搭建的应用不支持问答节点。
>
> - **＊**②：智能体发布 Chat SDK、微信小程序、抖音小程序、微信、飞书、多维表格、掘金时，不支持卡片效果。
>
> - ＊③：应用发布微信、飞书、多维表格、掘金时，不支持卡片效果。
>
> - 微信服务号、微信客服、微信订阅号不支持工作流输出节点提前输出消息内容，仅支持汇总后统一输出。
>
> - 微信小程序支持工作流输出节点提前输出。

> [!NOTE]
>
> **说明**
>
> - **＊**①：仅豆包白名单用户支持在豆包渠道上传图片和文件。如果需要此功能，你可以发送邮件至[豆包反馈邮箱](mailto:feedback@mail.doubao.com)或在豆包 APP 内单击**帮助与反馈**渠道提交咨询。
>
> - **＊**②：微信客服渠道不支持语音输入和开场白。
>
> - **＊**③：发布小程序后，仅支持展示全部预设问题，不支持展示部分预设问题。
>
> - **＊**④：当前仅豆包手机端支持语音通话，且不支持使用扣子的复刻音色，音色将无法生效。
