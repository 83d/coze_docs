# 扣子应用权限申请与使用情况说明

**应用权限申请与使用情况说明**

- 为保障“扣子”产品功能实现与安全稳定运行至目的，我们可能会申请或使用操作系统的相关权限；

- 为保障您的知情权，我们通过下列列表将产品可能申请、使用的相关操作系统权限进行展示，您可以根据实际需要对相关权限进行管理；

- 根据产品的升级，申请、使用权限的类型与目的可能会有变动，我们将及时根据这些变动对列表进行调整，以确保您及时获悉权限的申请与使用情况；

- 请您知悉，我们为业务与产品的功能与安全需要，我们可能也会使用第三方及关联方SDK，这些第三方也可能会申请或使用相关操作系统权限；

- 在使用产品的过程中，您可能会使用第三方开发的H5页面或小程序，这些第三方开发的插件或小程序也可能因业务功能所必需而申请或使用相关操作系统权限；

**安卓操作系统应用权限列表**

| **权限名称** | **权限功能说明** | **使用场景或目的** |
| --- | --- | --- |
| CAMERA 相机   | 用于拍摄图像   | 第三方网站为你提供特定服务时，可能需要通过扣子应用访问调用系统敏感权限，如“相机”权限，“位置”权限，“录音”权限   |
| LOCATION 位置   | ACCESS\_FINE\_LOCATION;访问精准定位 |  |
|  | ACCESS\_COARSE\_LOCATION 访问粗略位置 |  |
| RECORD\_AUDIO 录音 | 使用麦克风录制音频 |  |
| READ\_MEDIA\_IMAGES 读取图片 | 允许App从外部存储读取图片数据 | 用于帮助你完成创建智能体选择头像、发送图片等功能   |
| READ\_MEDIA\_VIDEO 读取视频 | 允许App从外部存储读取视频数据 | 用于帮助你完成创建智能体选择头像、发送视频等功能 |
| WRITE\_EXTERNAL\_STORAGE 写入外置存储器   | 允许App写入外置存储器   | 用于帮助你存储照片及下载文件等功能，适用于*AndroidSDK版本33 及以下版本*   |

| POST\_NOTIFICATIONS App Push | 向用户发送推送时 | 用于用户消息告知、身份验证等场景 |
