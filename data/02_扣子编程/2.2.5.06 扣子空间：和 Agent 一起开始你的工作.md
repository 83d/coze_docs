# 扣子空间：和 Agent 一起开始你的工作

## 什么是扣子空间

扣子空间是你和 AI Agent 协同办公的最佳场所。在扣子空间里，精通各项技能的「通用实习生」，各行各业的「领域专家」，任你选择。把任务交给扣子空间，把时间还给你自己。

## 扣子空间能做什么

扣子空间提供了强大的功能，全面提升生产力，支持多种应用场景，满足不同用户的需求。

- **生产力全面提升**：**从回答问题，到解决问题，让 AI Agent 帮你完成更多的工作。**

- 不局限于简单回答问题，更能切实帮你解决问题。无论是工作中的复杂任务，还是生活里的琐碎事务，扣子空间凭借强大的主动搜索，深度分析和精准执行，助你高效完成更多工作，节省时间与精力。

- **专家能力深度支持**：**专家 AI Agent 生态，让更专业 AI Agent 来为你提供服务。**

- 这里汇聚各领域专业的 AI Agent，它们就如同现实中的专家一样，具备深厚专业知识储备。无论是股票咨询还是用户研究等，都有对应的专业 AI Agent 来为你提供高质量服务，精准且靠谱。依托此生态，你能获取更具权威性的助力，让专业的 AI Agent 成为你前行路上的有力支撑。

- **协作模式灵活调整**：**探索/规划双模式，更好地和 AI Agent 一起协作完成高难度任务。**

- 有着独特的探索、规划双模式。你既可以选择让 AI Agent 自动处理任务，省心省力；也能开启规划模式，与它携手并肩。面对高难度任务时，这种灵活度能发挥大作用，充分结合你的智慧和 AI Agent 的优势，共同攻克难题，让协作更顺畅高效。

- **能力边界拓展延伸**：**MCP 扩展集成，无限拓展 AI Agent 能力边界。**

- 借助 MCP 扩展集成，为 AI Agent 开启了无限可能。传统的能力局限将不复存在，通过这一强大功能，不断拓展其能力边界，让 AI Agent 能涉足更多领域，应对更复杂多样的需求。无论你是探索未知知识，还是挑战高难度项目，它都能凭借拓展后的能力为你助力。

## 如何使用扣子空间

激活扣子空间后，即可开始使用扣子空间。具体操作，请参考[了解扣子](https://docs.coze.cn/cozespace/overview)。
