**扣子，用 Agent 重塑生产力。**扣子现已升级为**新一代 AI Agent 平台**，在「扣子开发平台」，你可以开发 AI Agent 和各类 AI 应用；「扣子罗盘」则会帮你轻松搞定运维和效果调优。让扣子，成为你和 AI Agent 的最佳相处之道。

扣子品牌升级，带来了全新的产品布局与服务优化，涵盖订阅套餐升级、两款产品的重磅上线以及官网首页的焕新改版。

### 扣子开发平台：零基础开启 Agent 专业开发

在全新的品牌定位下，扣子开发平台进一步优化升级，推出丰富的订阅套餐，为开发者带来更强大的功能和更灵活的使用体验。

#### 全新的品牌定位

扣子开发平台（原扣子）是一站式 AI Agent 开发工具。提供各类最新大模型和工具、多种开发模式和框架，从开发到部署，为你提供最便捷的 AI Agent 开发环境。上万家企业、数百万开发者正在使扣子开发平台。

- **提供 AI Agent 开发所需的全部核心技术**

- Prompt、RAG、Plugin、Workflow、UI Builder ，使得开发者可以聚焦创造 AI 核心价值。

- **开箱即用，用最低的成本开发最专业的 AI Agent **

- 扣子开发平台为开发者提供了健全的应用模板和编排框架，你可以基于它们快速构建各种 AI Agent ，将创意变为现实。

- 集成火山引擎各类资源，方便你的 AI Agent 实现快速扩容。

- **安全可信，多种方式提供全方位保障你的数据安全**

- 数据加密、VPC私网连接、PrivateLink 等多种方式，满足企业级的安全需求。

#### 丰富的订阅套餐

随着扣子品牌升级，为满足不同的开发需求，面向不同的开发人群，扣子开发平台推出**个人进阶版**、**团队版**以及**企业版**订阅套餐，采用包年包月+按量付费的混合计费模式。各个订阅套餐的权益范围不同，企业团队版可享受多人协作等高级权益。

需要注意的是，订阅套餐属于计费体系的重大升级，对原基础版及专业版用户的影响可参考[影响范围](https://docs.coze.cn/guides/coze_premium_announcement)。关于订阅套餐的详细说明，可参考[订阅套餐](https://docs.coze.cn/coze_pro/premium_package)。

### 扣子罗盘：让 Agent 开发调优不再迷航

扣子罗盘（CozeLoop）是 AI Agent 时代的全生命周期调优平台，为企业与开发者提供一站式智能运维解决方案，让 AI Agent 开发不再迷航。平台以数据驱动为核心，集成以下核心能力。

- **Prompt 智能中枢**：支持 Prompt 调试、版本管理、多版本对比及服务化调用，让 Prompt 开发更高效；

- **实时观测预警**：通过 Trace 数据追踪、性能统计、异常实时告警、智能打标、实时数据评测，快速定位问题根源；

- **全链路评测体系**：在 AI Agent 生命周期沉淀评测集，覆盖 AI Agent 端到端测试，精准衡量效果；

扣子罗盘以 “预警在先、调优在手、洞察在侧” 为理念，助力开发者专注创新，让 AI Agent 在复杂场景中稳定落地，为企业构建可靠、智能的决策能力。

访问[扣子罗盘](https://loop.coze.cn/console)，快速体验 AI 应用 DevOps 解决方案。

### Eino 框架：Agent 开发从未如此简单

Eino 是基于 Go 语言的 AI Agent 开源开发框架，提供了丰富的辅助 AI Agent 开发的原子组件、集成组件、组件编排、切面扩展等能力，可以帮助开发者更加简单便捷地开发出架构清晰、易维护、高可用的 AI Agent 和各类 AI 应用。

- **统一的组件抽象和丰富的实现**：Eino 将 AI 应用开发中常见的能力抽象为组件，统一的定义让 AI 应用能力变得简单清晰，每种组件抽象都有多种实现，开箱即用

- **灵活的编排能力**：提供 Graph、Chain、Workflow 多种编排模式，强大且灵活，用简单的方式表达复杂的业务逻辑，编排中还提供了类型检查、并发管理、切面管理、可扩展的 Option 等能力

- **完善的流处理能力**：根据输入和输出是否为流，组成 4 种请求交互模式： invoke、stream、collect、transform。Eino 提供了流的自动转换、流的合并、流的复制等能力

- **强大的工具链**：可视化的编排和调试工具，拖拽几下就可以完成一个 AI 应用构建。

关于 Eino 的详细说明，可参考 [Eino 概述](https://www.cloudwego.io/zh/docs/eino/quick_start/)。
