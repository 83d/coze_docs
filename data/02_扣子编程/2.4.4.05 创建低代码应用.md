# 创建低代码应用

本文指导你从一个空白的低代码应用开始。此外，扣子编程提供了低代码应用模板帮你快速上手，参考[使用低代码应用模板](https://docs.coze.cn/guides/create_app_from_template)。

## 背景介绍

扣子编程的低代码应用类似于 Visual Studio Code 中的 Project 概念。你可以使用工作流等能力完成业务逻辑编排，并通过扣子编程提供的可视化搭建能力完成用户页面搭建。

> [!NOTE]
>
> **说明**
>
> 在应用内单独创建的资源只能被这个应用引用。如果想要变成共享资源，需要先将其转移至空间下的资源库。

## 操作步骤

参考以下操作，创建低代码应用项目。

1. 登录[扣子编程](https://code.coze.cn/home)。

2. 在页面顶部选择目标工作空间，然后在左侧导航栏中单击**新建项目**。

3. 在**低代码模式**区域，将鼠标移动到**更多**，然后单击**打开应用开发**。

-

4. 在**应用模板**页面，单击**空白应用**。

5. 输入应用名称，并单击图标旁的 AI 图标使用 AI 自动生成一个图标。然后单击**确定**。

- 应用创建成功后，你会直接进入到应用的集成开发环境 (IDE)。
