# 快捷键

用户界面编辑器支持快捷键操作，以提升搭建用户界面的效率。

> [!NOTE]
>
> **注意**
>
> 目前，用户界面编辑器不支持自定义设置快捷键。为了避免与你使用的其他应用程序的快捷键发生冲突，建议你提前调整这些应用程序的快捷键配置，以确保用户界面编辑器的使用体验不受影响。

| **操作** | **mac 快捷键** | **windows 快捷键** | **说明** |
| --- | --- | --- | --- |
| 复制 | command + c | ctrl + c | 支持通过快捷键复制组件。复制组件时会复制组件及其所有属性，复制容器类组件，会复制容器类组件及其子组件。  复制组件后，系统将根据用户后续的鼠标操作确定粘贴位置，具体如下：  - 如果鼠标未点击选中其他组件，会将复制的组件粘贴到同级位置。  - 如果鼠标单击选中了另一个容器类组件，会将复制的组件粘贴到选中的容器类组件中。  - 如果鼠标单击选中了另一个非容器类组件，会将复制的组件粘贴到选中组件的同级位置。 |
| 粘贴 | command + v | ctrl + v | 将剪贴板中的组件或页面插入到指定的位置。 |
| 撤销 | command + z | ctrl + z | 撤销最近一次或几次操作。  如果执行撤销操作以回溯至历史记录中的某个状态，并随后执行新的操作，系统将这些新操作视为一个新的历史分支。因此，无法通过重做操作返回至之前撤销过的操作状态。 |
| 重做 | command + shift + z | ctrl + shift + z | 重新执行刚刚被撤销的操作。 |
| 剪切 | command + x | ctrl + x | 剪切操作会移动选中的组件到剪贴板，同时从原始位置删除该组件。 |
| 创建副本 | command + d | ctrl + d | 为选中的组件快速创建副本。 |
