# 智能绘图_文生图插件

智能绘图\_文生图插件

[智能绘图\_文生图插件](https://www.coze.cn/store/plugin/7516843480623382568?from=add_plugin_menu)搭载了 Seedream 通用 3.0 文生图模型，用于根据输入的文本描述生成全新的图像。Seedream 通用 3.0 文生图模型对语义有更好的理解，实体结构也更加自然真实，支持输出 2048 分辨率图片，并在各类场景下均展现出显著的提升效果。

## 使用限制

扣子主账号内所有子账号共享**智能绘图\_文生图插件**的并发限制，其值为 7。

## 计费说明

智能绘图\_文生图插件根据插件调用次数计费，对应的计费项及单价请参考[插件费用](https://docs.coze.cn/coze_pro/plugin_fee)。

## 配置说明

调用 genImage 工具时，你需要输入生成图像的提示词，用于指定生成图片的主题。此外，你还可以进一步指定随机种子，以确保在相同的输入条件下生成一致的图像；调整文本描述的影响程度以控制提示词对生成图像的影响力，从而生成更符合需求的图像。

### 输入参数

输入参数说明如下表所示：

| **参数** | **说明** |
| --- | --- |
| prompt | 设置用于生成图像的提示词。支持中英文。必填参数。 |
| logo\_info | 设置水印信息，Object 类型，包括如下字段：  - add\_logo：是否添加水印，Boolean 类型。取值：  - false（默认值）：不添加水印。  - true：添加水印，设置为 true 时，如下参数才生效。  - language：设置默认水印，Integer 类型，取值：  - 0（默认值）：中文，内容为AI生成。  - 1：英文，内容为Generated by AI。  - logo\_text\_content：自定义的水印内容。如果设置了 logo\_text\_content 参数，那么 language 的配置不生效。  - opacity：水印的不透明度，取值范围为 0～1，1 表示完全不透明，默认值为 0.3。  - position：水印的位置，取值：  - 0（默认值）：右下角  - 1：左下角  - 2：左上角  - 3：右上角  配置示例如下：   {  "add\_logo": true,  "language": 0,  "logo\_text\_content": "水印内容",  "opacity": 0.3,  "position": 0  }  |
| scale | 设置文本描述对图像生成的影响程度，默认值为 2.5。 |
| seed | 设置随机种子，输入 -1 或正整数。  - -1（默认值）：系统将随机生成图片，每次生成结果不同。  - 任意正整数：系统将基于该种子值生成相似的图片，确保结果可复现。 |
| use\_pre\_llm | 选择是否开启文本扩写。  - true：开启扩写功能，即系统会扩写及优化输入的提示词。  - false：关闭扩写功能。 |
| width | 设置图像生成后的宽度，默认值为 1328。 |
| height | 设置图像生成后的高度，默认值为 1328。 |

### 输出参数

输出参数说明如下表所示：

| **参数** | **说明** |
| --- | --- |
| log\_id | 日志 ID。 |
| msg | 执行插件时的状态描述或错误提示信息。 |
| code | 执行插件时的状态码。 |
| data.image\_urls | 生成的图片 URL，目前只支持生成一张图片。图片有效期为 30 天，建议及时转存。 |
| data.request\_id | 请求 ID。 |
| data.binary\_data\_base64 | 输出处理过的图片的 Base64 编码。 |
| data.algorithm\_base\_resp.status\_code | 子状态码，进一步区分错误原因。 |
| data.algorithm\_base\_resp.status\_message | 子状态信息，进一步区分错误原因。 |
