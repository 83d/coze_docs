# 计费常见问题

本文介绍扣子常见的计费问题。

## 费用相关

### 什么情况下会扣除账号余额，并导致欠费？

购买企业版后，当扣子账户内的积分不足时，会根据消耗的积分对应价格从现金账户中扣费。另外，在扣子平台使用火山方舟、火山知识库等其他火山产品或付费技能、付费模板、三方付费插件时会涉及相关费用，这些费用需从账户余额中扣除。

### 企业版还有积分，为什么会扣款导致欠费？

如果你发现企业版账号内仍有积分，但出现扣款导致欠费，通常是因为在扣子中使用了火山方舟、火山知识库等火山产品产生了其他相关费用，这部分费用不支持通过扣子积分抵扣，将直接从账户余额中扣除。例如在火山方舟创建了模型接入点，在扣子中使用方舟模型，火山方舟模型服务产生的 Token 费用，由方舟服务收取，从账户余额中扣除。

### 如何设置仅消耗积分，避免欠费？

个人版不支持超额使用，积分为 0 后，无法继续开发或运行项目。企业版不支持设置仅消耗积分，但你可以开启余额预警，以便及时充值避免欠费。具体操作，参考[积分预警](https://docs.coze.cn/coze_pro/credits)。

企业版的积分用完后将从现金账户扣费（存在1～2小时出账延时），当现金账户余额不足时，企业版账号转为欠费状态并停止服务。

### 收到欠费短信，如何查看具体原因？

购买企业版后，你可以在火山引擎费用中心的[账单明细](https://console.volcengine.com/finance/bill/detail/)页面查看消费记录。扣子计费项较多，用量统计和出账可能存在延迟。具体操作，请参考[账单管理](https://www.volcengine.com/docs/6269/94010)。

### 为什么欠费关停后，仍收到了账单？

由于企业版账单存在延时，并且只有在出账扣款操作完成后，企业版账户才会进入欠费关停状态，在此之前你仍可以使用扣子，并可能产生费用，因此你可能在服务关停后仍收到账单。例如在 16:00～17:00 期间使用扣子产生了费用，系统可能在 17:30 才出账并扣费，在 17:00 ～ 17:30 期间，你仍可以使用扣子，但实际 16:00～17:00 期间的费用已导致欠费。17:00 ～ 17:30 期间的费用会计入 17:00 ～ 18:00 的账单中，你将在欠费关停之后收到 17:00 ～ 18:00 时间段的账单。

### 如何停止计费？

1. 在扣子编程中删除智能体、工作流、应用、插件、音色等所有资源，次日开始停止计费。

2. 如果曾经使用过其他火山产品，例如火山方舟模型、火山知识库等，需要删除方舟接入点、知识库等火山侧的资源，否则这些资源将继续产生费用。

如果你后续不再使用扣子，也可以直接删除账号下的所有资源，并注销扣子账号。详细操作步骤可参考[注销账号](https://docs.coze.cn/guides/cancel_account)。

### 如何设置积分余额提醒？

购买企业版后，你可以在[火山引擎扣子控制台](https://console.volcengine.com/coze-pro)**概览**页面，设置积分余额预警。如需修改通知人信息和接收方式，请前往[消息中心](https://console.volcengine.com/message/person)进行配置。具体操作，请参考[积分预警](https://docs.coze.cn/coze_pro/credits)。

### 如何设置账单消费预警？

你可以在火山引擎费用中心的[账户总览](https://console.volcengine.com/finance/account-overview/)页面，开启账单消费预警。具体操作，请参考[设置账单消费预警](https://www.volcengine.com/docs/6269/942364)。

### 如何开具发票？

- 个人版：扣子个人版发票由扣子开具，仅支持开具增值税普通发票。具体操作及开票规则，请参考[开具发票](https://docs.coze.cn/coze_pro/Issue_invoice)。

- 企业版：扣子企业版发票由火山引擎开具，**发票抬头需与账号实名认证主体保持一致**。

- 个人抬头发票：请打开[实名认证](https://console.volcengine.com/user/authentication/)页面，完成个人实名认证。

- 企业抬头发票：先完成企业认证，具体操作请参考[企业认证](https://www.volcengine.com/docs/6261/64937)。

- 完成认证后，可以在火山引擎费用中心的**[发票管理](https://console.volcengine.com/finance/invoice)**页面，申请开具发票。具体操作及开票规则，请参考[开票操作指引](https://www.volcengine.com/docs/6269/65297)。

### 如何申请扣子产品退订、退款？

- **个人版套餐**：支持退订，不支持退款。

- **连续包年、包月的用户**：支持随时申请退订，即关闭自动续费功能。退订将在**当前订阅周期结束后**正式生效，当前订阅周期内你仍可以享有套餐权益。例如退订连续包年的个人高阶版后，退订将在下一年度生效。具体操作，请参考[订阅管理](https://docs.coze.cn/coze_pro/personal_plan)。本操作不涉及退款，你在购买时仅支付当前订阅周期内的费用。

- **单月购买用户**：无需退订，一个月到期后，自动恢复到个人免费版。

- **企业版套餐**：你可以通过火山引擎费用中心[退订管理](https://console.volcengine.com/auth/login?redirectURI=/finance/unsubscribe)页面操作退订。退款规则如下：

- **无理由全额退款权益**：每个账号在自然年内享受一次“购买 7 天内且未使用无理由全额退款”的权益（增购积分、购买企业版等总计仅可使用一次）。

- **非无理由退款规则**：不符合7天未使用的无条件退款规则时，将按以下规则计算退款金额。

- 企业版的版本和积分费用将分开计算退款金额，具体说明，请参考[退订企业版](https://docs.coze.cn/coze_pro/enterprise_plan)。

- 增购积分不支持退款。

### 对公转账汇款后，账户余额未更新怎么办？

当你采用对公汇款方式对火山引擎账户进行充值时，如果确认已正确填写汇款信息，但未查询到认款的流水，可能是由于银行处理原因，该笔流水还未到火山账户，请稍等 1～2 小时重新查询。

关于对公转账汇款的更多信息，请参考[充值操作指引](https://www.volcengine.com/docs/6269/100434)。

## 积分相关

### 套餐到期后，账户里的积分还能用吗？

- **个人版**：套餐到期后，你的订阅积分将失效，账户中的活动积分与增购积分可以继续使用。如果你重新订阅，将在新订阅周期首日获得订阅积分。

- **企业版**：套餐到期后，所有积分将失效。如果你重新订阅企业版，将在新订阅周期首日获得订阅积分，并可继续使用有效期内的增购积分。

### 增购积分支持退款吗？

- 个人版增购积分不支持退订、退款。

- 企业版每年享受一次“购买 7 天内且未使用无理由全额退款”的权益。不满足时，不支持退订。更新信息，请参考[退订规则](https://docs.coze.cn/coze_pro/enterprise_plan)。

### 存量的扣子资源包还能用吗？

根据你当前使用的套餐版本，存量资源包的处理方式如下：

- 企业版用户：已购买的扣子资源包、Lite 资源包可继续使用，抵扣范围保持不变。

- 个人版用户：由于个人版的支付结算链路已从**火山引擎**迁移至**扣子**，因此从原付费套餐升级到新个人版时，存量的资源包会被自动退订、退款。

### 资源点和积分的关系？

2026 年 1 月 18 日起，扣子**资源点**更名为扣子**积分**。仅为名称变更，积分的抵扣规则、范围不变。

## 用量相关

### 如何查看积分使用情况？

你可以在[扣子编程](https://code.coze.cn/home)的**积分消耗**页面，查看月度积分总消耗以及各个扣子、扣子编程任务的积分消耗情况。具体操作，请参考[查看积分消耗明细](https://docs.coze.cn/coze_pro/credits)。

如果你购买的是企业版，以[账单明细](https://console.volcengine.com/finance/bill/detail)为准，并建议设置账户余额预警，避免积分消费完毕，开始抵扣账户内的现金余额。具体操作，请参考[如何设置积分余额提醒？](https://docs.coze.cn/coze_pro/billing_faq)。

### 如何查看每次编程任务消耗的积分？

在开发 AI 编程项目时，你可以在对话区域，将鼠标移动到对应对话结束位置右下角的空白区域，查看单次对话消耗的积分。

### 任务开发失败，如何抵扣积分？

因为扣子平台技术问题而失败的任务，平台将进行周期性补偿。系统自动识别并处理，无需手动申请。更多信息，请参考[失败任务补偿积分](https://docs.coze.cn/coze_pro/credits)。

### 扣子客服智能体产生的 Token 属于大语言模型推理服务费用吗？

正常使用扣子客服智能体时产生的 Token 费用属于大语言模型的推理服务输入输出费用，详情请参考[模型费用](https://docs.coze.cn/coze_pro/model_fee)。

### 是否支持为不同的智能体分配定额用量？

目前，扣子不支持为不同的智能体分配定额用量。如需了解平台的用量统计详情，请参考[查看积分消耗明细](https://docs.coze.cn/coze_pro/credits)。

### 如何查看每轮对话消耗的计费 Token？

你可以在智能体**预览与调试**页面的对话记录中，查看每轮对话消耗的计费 Token。具体费用，以[账单](https://console.volcengine.com/finance/bill/detail/)为准。

## 计费方式相关

### 调用智能体如何收费？

在扣子编程中或其他发布渠道调试或测试智能体时，根据使用的功能收取以下费用：

- 默认收取模型费用，具体价格和模型类型、版本和 Token 数量有关。

- 如果使用了付费插件，则根据插件调用量收取插件费用。

- 如果通过 Realtime SDK、RTC SDK 调用智能体，根据音视频用量收取**实时音视频**费用；如果对话中使用了语音识别或语音合成 API，根据语音服务的用量额外收取**智能语音**费用。

### 调试智能体收费吗？

在扣子编程中在线调试智能体时，根据使用的功能收取以下费用：

- 默认收取模型费用，具体价格和模型类型、版本和 Token 数量有关。

- 如果使用了收费插件，则根据插件调用量收取插件费用。

### 调用工作流如何收费？

在扣子编程中，调用工作流本身不会产生费用，但如果在工作流中添加了需要付费的大模型节点或插件节点等资源时，会产生对应的费用。

### 调用 API 或 Chat SDK 如何收费？

- 通过 HTTP 请求或 SDK 调用[发起对话](https://docs.coze.cn/developer_guides/chat_v3)等涉及模型交互的 API 时，根据 Token 量收取模型 Token 费用。

- 调用语音合成、语音识别等语音 API 时，根据语音服务的用量收取**智能语音**费用。

- 使用 Realtime SDK、RTC SDK 时，根据音视频用量收取**实时音视频**费用。

- 调用[查看对话详情](https://docs.coze.cn/developer_guides/retrieve_chat)等不涉及模型和语音、视频的 API 不会产生任何费用。
