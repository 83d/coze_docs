# 专业版计费规则

本文介绍原扣子专业版的计费项、资源限额、欠费与关停等计费规则。

> [!NOTE]
>
> **说明**
>
> - 扣子专业版不再支持新购，已注册专业版用户可继续使用。如果选择升级到新的订阅套餐，具体的影响请参考[影响范围](https://docs.coze.cn/guides/coze_premium_announcement#0156ffd6)。
>
> - 为了推动用户体验的一致性，提升产品的迭代效率，原专业版将于 2026 年 5 月 30 日正式下线，届时所有专业版用户将自动调整为个人免费版，建议所有专业版用户尽快升级到新的订阅套餐。更多信息，请参考[扣子订阅套餐升级公告](https://docs.coze.cn/guides/20260119_coze_premium_upgraded)。

## 什么是扣子专业版？

在推出订阅套餐前，扣子专业版主要面向对稳定性和用量有高需求的专业开发者，支持较大的工作空间规模、免费知识库空间以及协作编辑容量，付费功能保障专业级 SLA，不限制调用请求频率和总量，费用按实际用量计算。

## 免费额度

除了每日 500 积分的免费使用额度之外，使用豆包模型、调用 API、Chat SDK、使用收费插件均有一定免费额度。

| **类型** | **说明** |
| --- | --- |
| 模型免费额度  | - 扣子模型中仅豆包模型、 DeepSeek 模型和 Kimi-K2-250905 模型收费。使用 **Kimi-8k**、**Kimi-32k**、**Kimi-128k**、**阶跃星辰**等扣子提供的其他模型暂不收取费用，但每日调用次数有一定限制。费用说明请参考[模型费用](https://docs.coze.cn/coze_pro/model_fee)。  - 扣子调试区、扣子商店、模板、工作流调试、API 等豆包以外的所有发布渠道正常扣减。 |
| API 免费额度 | 每个账号累计免费调用 500 次 API。超过次数后根据模型 Token 量扣减。 |
| Chat SDK 免费额度 | 每个账号累计免费调用 500 次 Chat SDK。超过次数后根据模型 Token 用量扣减。 |
| 收费插件 | 部分付费插件有一定免费额度，额度耗尽后如需继续使用，会从账号中扣减。费用说明请参考[插件费用](https://docs.coze.cn/coze_pro/plugin_fee)。 |

## 计费项

扣子为专业版用户提供每日 500 免费积分。免费积分使用完毕后，专业版用户可购买积分或充值一定金额以付费使用扣子。使用扣子专业版过程中，会产生如下计费项。

- [模型费用](https://docs.coze.cn/coze_pro/model_fee)

- [插件费用](https://docs.coze.cn/coze_pro/plugin_fee)

- [音视频费用](https://docs.coze.cn/coze_pro/asr_tts_fee)

- [知识库空间费用](https://docs.coze.cn/coze_pro/knowledge_storage_fee)

- [智能体调用费用（已下架）](https://docs.coze.cn/coze_pro/bot_request_fee)

## 欠费

### 欠费说明

开通专业版后，当你的账户内的积分不足时，会根据消耗的积分对应价格从现金账户中扣费。

另外，专业版支持通过火山引擎方舟大模型服务平台接入模型资源，如果在火山方舟创建了模型接入点，在扣子中使用方舟模型，火山方舟模型服务产生的 Token 费用，由方舟服务收取。这部分费用不支持通过积分抵扣，将直接从账户余额中扣除。

### 欠费状态

- **欠费触发**：当现金账户余额不足以支付按量计费产生的账单时，扣子账号会转为欠费状态，你需要及时补缴欠款。

- **出账延时与欠费**：当前出账系统存在延时，并且只有在出账扣款操作完成后，账户才会进入欠费状态，在此之前你仍可以使用扣子，并可能产生费用。例如 16:00～17:00 期间产生的费用，可能在 17:30:00 才出账。如果是 16:00～17:00 期间的费用导致欠费，系统将 17:30:00 进行扣款，届时账号进入欠费状态。

### 欠费影响

- 账号欠费后存在关停风险，为了避免影响业务运行、对服务造成影响，请及时充值、结清欠费账单。具体操作，请参见[充值操作指引](https://www.volcengine.com/docs/6269/100434)。

- 账号欠费后各个订阅套餐均不可用。为避免影响正常使用，请及时补齐欠款。

### 预防措施

建议在[费用中心](https://console.volcengine.com/finance/account-overview)设置**余额阈值提醒**，实时关注账户余额情况。你也可以在[费用中心](https://console.volcengine.com/finance/account-overview)查看账号余额，详情请参考[如何查看可用余额](https://www.volcengine.com/docs/6269/100429#板块说明)。

## 关停

专业版欠费 2 小时后，扣子账号会转为欠费关停状态。此状态下只能查看资源，不能编辑或使用资源。

关停期间，扣子账号中的智能体、知识库等资源将会持续保留，但系统将自动拦截扣子所有可能产生资源消耗的请求，包括但不限于智能体调用、API 请求等，你将无法使用任何与扣子相关的服务，直至所有欠款补缴完毕。

> [!NOTE]
>
> **注意**
>
> 关停后，超额的知识库空间仍会继续产生费用，直到你删除知识库或知识库被系统回收。

### 遇到“当前空间为扣子专业版专用空间，不支持组织外部的成员加入”错误时，如何处理？

扣子专业版的团队空间仅支持邀请火山引擎账号下的子账户加入团队空间，请确认添加的空间成员是否为当前账号下的子用户。

### 专业版如何切换为个人免费版？

专业版不支持直接切换为个人免费版。如果你确认要回退，可以升级到个人进阶版（最低可购买一个月），个人进阶版到期后，会自动回退到个人免费版，后续可以免费使用扣子。

注意，个人版均不支持子用户、火山模型等一系列高级权益，为避免影响你已创建的子用户或使用火山模型的智能体，建议你维持专业版或者升级为企业版

### 如何升级专业版企业版？

1. 专业版主账号购买企业版，然后创建企业。具体操作，请参考[创建企业或团队](https://docs.coze.cn/guides/create_team_and_enterprise)。

2. 企业的超级管理员将工作空间迁移到企业版。具体操作，请参考[迁移空间到团队或企业](https://docs.coze.cn/guides/enterprise_workspace#fdef76af)。

3. 企业的超级管理员，将存量的专业版子用户加入到企业中。具体操作，请参考[加入企业](https://docs.coze.cn/guides/create_member#0ec2a393)。

- 企业版购买者会自动成为企业的超级管理员。

### 升级专业版到企业版会有什么影响？

升级专业版到企业版，受到的影响范围如下：

- 需要按月或按年购买订阅套餐，到期后会回退到个人免费版权益，无法回退至专业版。

- 取消智能体调用费。

- 升级后，存量的子用户默认只享受个人免费版权益，需要由企业的超级管理员或管理员手动将子用户加入到企业中，以享受企业版权益。具体操作，请参考[加入企业](https://docs.coze.cn/guides/create_member#0ec2a393)。

### 专业版升级到企业版，已经创建的子用户如何处理？

专业版升级到企业版后，企业管理员需要将子用户添加为企业成员，并支付成员费。没有被添加为成员的子用户，只能在个人账号里使用，且个人账号为免费版权益。但为了不影响子用户线上已发布的资源，所有子用户在个人账号下所消耗的积分依然会扣减主账号的积分余额。
