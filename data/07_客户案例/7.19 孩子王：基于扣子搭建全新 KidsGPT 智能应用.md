# 孩子王：基于扣子搭建全新 KidsGPT 智能应用

孩子王基于扣子编程搭建了全新的 KidsGPT 智能育儿顾问。在原先智能问答功能的基础上，KidsGPT 智能育儿顾问新增了商品推荐、孕育通识、情感互动与活动推荐四大能力。同时，它还具备多轮对话中的长期记忆功能，能够以更人性化的方式与客户交互，精准解答客户在选购商品或服务时的复杂需求。

## 客户介绍

孩子王是一家以数据驱动、基于客户关系经营的创新型亲子家庭服务商，专业为准妈妈及 0-14 岁儿童提供全渠道一站式商品解决方案、育儿成长及社交互动服务。KidsGPT 智能育儿顾问则是孩子王在智能化服务领域的一次重大创新，能够为客户提供智能化、个性化的育儿服务。

## 业务挑战

孩子王作为一家创新型亲子家庭服务商，致力于为广大父母提供全方位的育儿支持和优质的产品服务。然而，在实际服务过程中，面临着诸多业务挑战。

- **复杂需求解答**：客户在育儿过程中往往面临复杂多样的问题，尤其是在商品选购和服务咨询方面，传统的智能问答系统无法深入理解用户的具体需求，更无法提供个性化的建议，导致客户体验不佳，需要进一步的人工客服介入。

- **数据整合与利用**：孩子王积累了大量的客户数据，包括购买历史、咨询记录、客户偏好等。然而，这些数据缺乏有效的整合和分析。如何将这些数据转化为有价值的洞察，以优化产品推荐、提升服务质量和个性化体验，是孩子王面临的一个重要挑战。

## 解决方案

针对孩子王特有的商业化场景，孩子王借助扣子编程搭建了 KidsGPT 智能育儿顾问，重点围绕以下五个方面展开：

<img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/9d342ed02e764f9a8f1711bf339252a4~tplv-goo7wpa0wc-quality:q75.awebp" width="508">

- **数据质量把控**：数据是 Agent 工作的核心基础。以奶瓶、奶粉、纸尿裤等品类为例，对数据进行精细化整理，对缺失数据查漏补缺，规范异常数据（如规范适用年龄段），清洗重复商品详情等，以实现去噪操作。同时，整合数据索引、数据集、商品品类及通用知识，建立完善的数据版本控制体系，确保最大程度提升 Agent 的回答质量。

- **构建 AI 基础设施**：打造企业级 AI 基础建设，包括实时动态数据和静态的企业级知识库，整合专业知识信息以提升问题回答的准确度。搭建知识搜索引擎，依托先进的 AI 搜索能力，提高商品推荐的专业度。另外，建立多 Agent 系统，结合多 Agent 能力，有效扩展 Agent 的能力边界。

- **意图识别与多 Agent 协同**：通过意图识别和多 Agent 系统的协同工作，实现智能化服务。例如情感互动 Agent 结合大模型和会员系统等，根据节日、客户生日等时间点，主动发送祝福和关怀；商品推荐 Agent 依据历史购物信息、多轮对话以及公司销售战略，精准推荐商品。多个 Agent 相互配合，满足顾客在不同场景下的需求。

- **插件集成**：基于扣子编程丰富的插件，扩展 Agent 的能力边界，例如，通过联网搜索、文生图、发邮件等功能，使 Agent 具备多模态交互能力、完善的知识库以及丰富的插件生态系统。其核心优势在于降低开发门槛，提升开发效率。

- **评测集与管理**：由于生成式大模型可解释性差，线上问题的自动化评测至关重要。通过对单轮对话、多轮对话、长上下文等场景进行测试与评测，涵盖场景分类、问题分类等多个维度。基于评测结果，不断迭代优化 Agent，使其在 AI 销售领域达到 90% 以上的准确率。
