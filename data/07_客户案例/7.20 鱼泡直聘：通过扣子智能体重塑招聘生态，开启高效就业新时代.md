# 鱼泡直聘：通过扣子智能体重塑招聘生态，开启高效就业新时代

鱼泡直聘借助扣子编程智能体与工作流的深度融合，实现招聘全流程 Agent 接管，结构化完善率提升 60%、合规处理效率提升 90%；针对 Prompt 管理痛点，进一步用扣子罗盘搭建全生命周期管理体系，最终达成解决方案稳定性升 50%、Prompt 迭代效率升 80%、团队协同效率翻倍的成果，为招聘行业 AI 化提供高效范本。

## 客户介绍

鱼泡直聘成立于2017年，是中国领先的招聘服务平台，全国综合排名前四，在蓝领招聘市场保持第一，白领市场增速显著。依托大数据、AI 算法等数字科技，构建以鱼泡直聘为核心，涵盖鱼泡直聘校招、SaaS 工具等产品的数字生态矩阵，致力于提供全场景智能化招聘服务解决方案。

## 业务挑战

在引入扣子编程智能体之前，鱼泡直聘采用前端用户手动选择岗位信息，平台侧审核团队进行人工审核的流程，面临以下问题和痛点：

- **岗位结构化招聘方录入完成率低**

- 由于设置岗位分类繁多，用户在前台需手动选择的标签也会很多，造成一定程度上的效率浪费，增加了发布难度。

- **岗位合规审核依赖人工，效率低、速度慢**

- 考虑到招聘方的用人需求以及招聘岗位规范，在岗位发布时可能会使用到不合规的词语或内容。其涉及到的敏感词、极限词，特别是违规信息以及虚假招聘，对人工审核造成很大压力。

- 一些招聘方为了吸引求职者的关注，会夸大招聘效果，也会发布夸张高薪高待遇的内容。甚至包含限制性字样，例如“做满 X 天有工资”这样的限制字样，需要人工介入辅助审核。

## 解决方案

### 扣子编程接管招聘流程

<img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/f16da5eefc61460880c1daec15227a68~tplv-goo7wpa0wc-quality:q75.awebp" width="612">

鱼泡直聘通过扣子编程智能体与工作流的深度融合，实现了全流程的 Agent 接管，显著提升了招聘流程的效率与质量。具体解决方案如下：

- **岗位拆分**：将招聘方发布的包含多个岗位的招聘简章，智能拆分为独立的岗位。

- **岗位合规**：智能识别信息中潜在的违反劳动法、敏感信息等，提升岗位质量。

- **岗位结构化**：对岗位信息进行结构化标签处理，一方面提升招工方的发布便捷性，同时提升更加精准的匹配效果。

### 扣子罗盘实现 Prompt 体系化管理

<img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/ba5cea7859fe4c20a9fe1c890aea7b36~tplv-goo7wpa0wc-quality:q75.awebp" width="449">

通过扣子编程搭建的自动化招聘流程期间，研发团队发现扣子编程智能体、工作流等解决方案的 Prompt 更新与迭代等存在管理困难、迭代效率低、稳定性差等风险：

- **管理机制缺失**：Prompt 完全融合在智能体、工作流内，缺乏统一的管理和管控方式。

- **迭代成本高昂**：每次版本迭代、场景扩张、模型升级、生产级 Prompt 迭代时，需要大量的资源和人力投入线上保障。

- **评估体系匮乏**：提示词缺乏体系化评估方式，完全依赖提示词专家经验，稳定性问题频发。

为了应对上述 Prompt 管理与迭代的核心痛点，鱼泡直聘接入扣子罗盘，构建了覆盖 Prompt 全生命周期的标准化管理体系，实现从开发、评估到部署迭代的全流程高效管控，具体解决方案如下：

- **Prompt 版本管理**：依托统一的 Prompt 开发模块，对 Prompt 的设计、开发与迭代过程进行规范化规划和集中化管理，打破其与智能体、工作流的耦合壁垒。

- **Prompt 评测体系**：

- 构建全面、高质量的科学化评估器，结合评测集对高质量数据集的精细化管理，为 Prompt 迭代优化提供明确方向。

- 通过评测实验搭建集评测集、Prompt、评估器于一体的全自动高效评估机制，实现 Prompt 效果的快速验证。

- 支持在扣子编程工作流中高效完成 Prompt 引用与更新提交，最终达成 Prompt 的统一规划、高效迭代与稳定生产，实现生产级 AI Agent 全生命周期管理。
