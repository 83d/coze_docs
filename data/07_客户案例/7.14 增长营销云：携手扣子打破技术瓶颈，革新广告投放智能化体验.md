# 增长营销云：携手扣子打破技术瓶颈，革新广告投放智能化体验

增长营销云是字节营销中台推出的广告投放系统，营销云基于扣子编程搭建了营销 AI 助手，智能统计分析拉新和留存数据，根据自然语言生成并执行广告投放任务，实现复杂投放流程的自动化。

## 客户介绍

增长营销云是字节营销中台推出的自动化广告投放系统，旨在帮助业务用户高效管理广告投放流程。用户可以通过平台进行广告投放，同时利用 AI 进行数据查询、任务管理和优化建议。平台集成了 AI 问答助手，使用户能够通过自然语言交互完成广告投放相关的任务，如查看特定维度数据、创建特定任务（如克隆广告）等。

## 业务挑战

增长营销云广告投放平台的智能化升级，旨在通过用户问答式交互完成“看数、建任务、广告克隆”等复杂操作。然而，该功能的实现涉及多系统调用、数据处理及任务状态管理等多个模块，传统开发模式面临以下挑战：

- **开发效能瓶颈**：需投入大量人力，灵活性和迭代速度受到明显限制。

- **技术栈局限**：开发团队需深度介入代码逻辑、API 调用及变量处理等工作，传统对话框架对底层能力过度封装，缺乏灵活性。

- **模型接入困难**：希望通过自有 API Key 调用 GPT 模型以控制成本，但当前平台无法灵活配置模型接入方式。

## 解决方案

为解决广告投放系统的技术瓶颈，增长营销云团队引入了扣子编程作为核心工作流编排解决方案。扣子编程不仅具备强大的工作流编排能力，还拥有成熟的插件生态，包括由 ISP 团队开发的支持流式输出、数据同步等关键功能的 Link Agent 插件。

基于扣子编程搭建的广告投放 AI 问答助手实现了智能化对话管理、高效任务处理等能力，具体搭建方案如下：

1. **解构业务流程**

- 对广告投放工作流的关键环节进行拆解，包括用户查询数据、创建任务、调整广告等。

2. **设计工作流架构**

- **主流程**：梳理投放任务的核心流程，包括用户意图解析、数据查询、任务创建等关键步骤，确保投放任务的主要流程清晰、高效且有序推进。

- **大模型节点**：设置不同功能的**大模型**节点，用于处理多样化的任务，例如查询任务、执行任务、广告克隆等，提升任务处理的针对性和效率。

- **代码节点**：添加代码节点负责 API 调用、数据预处理、参数拼接等技术实现，为整个工作流的顺畅运转提供有力的技术支撑。

3. **集成插件**

- **Link Agent 插件**：集成 Link Agent 插件，实现对话管理、流式输出、状态同步等功能，提升交互的流畅性与即时响应性。

- **数据接口调用**：对接 Zebra API 等内部数据接口，及时获取业务数据，为工作流各个环节提供数据支撑。

4. **优化交互体验**

- 流式输出技术应用：采用流式输出技术，实时反馈任务状态，让用户随时掌握广告投放进展，极大提升用户体验。

- 知识库检索：搭建专属的知识库，当用户提问时，系统能够在第一时间从知识库中进行检索并提供答案，减少用户等待时间，提高问题解决效率。

- KPI 评估体系：建立以任务提交率为核心的 KPI 评估体系，为持续优化提供量化依据。

## 客户收益

基于扣子编程开展工作流编排工作，增长营销云团队在广告投放 AI 助手的建设过程中取得了显著成效。

- **任务执行效率提升**：采用自动化流程后，系统能够快速解析用户意图并执行任务。以任务提交率作为核心衡量指标，有力保障任务创建和执行顺畅，提升整体工作效能。

- **开发协作优化**：扣子编程提供的插件和工作流机制，赋能不同团队可独立开发子工作流，随后无缝集成到主工作流，极大提升开发协作效率。

- **用户体验升级**：使用 Link Agent 插件实现流式输出，用户可实时获取任务执行的中间反馈，避免长时间等待。同时借助智能诊断和 AI 任务生成功能，全方位提升用户使用体验 。
