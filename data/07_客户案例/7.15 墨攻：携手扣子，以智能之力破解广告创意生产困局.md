# 墨攻：携手扣子，以智能之力破解广告创意生产困局

墨攻是字节营销中台推出的营销管理系统，接入了扣子编程的创意助手智能体，智能搜索热门广告素材、召回抖音热门授权泛内容视频，用于素材剪辑或一键成片。

## 客户介绍

墨攻是字节营销中台对外的一套综合营销管理系统，旨在为广告投放、素材管理、数据分析等环节提供高效的解决方案。作为广告运营的重要支撑，墨攻通过智能化和自动化技术，显著提升广告投放的精准度，同时优化广告创意的生产流程，助力营销效率最大化。

## 业务挑战

传统的广告创意生产流程通常依赖大量人工搜索、分析和制作，效率低下且难以保障创意质量。

- 前期素材筹备阶段，工作人员往往需要耗费大量时间与精力，通过各类搜索引擎、数据库以及线下资料进行人工搜索。这一过程不仅繁琐，还极易因个人主观局限而遗漏关键信息。

- 分析阶段，主要依靠人工对搜索到的素材逐一研判，从市场趋势、受众喜好到品牌调性匹配度等多方面进行考量，受限于人力分析的广度与深度，很难全面。

## 解决方案

基于上述业务挑战，墨攻团队通过扣子编程搭建了一整套智能化的广告创意支持系统。具体实现方式如下：

- **构建创意助手**

- 通过扣子编程提供的工作流功能，搭建**创意助手**智能体。通过在大模型节点进行精心微调的 Prompt，实现了自然语言与 SQL 语句的高效转换，支持广告素材的智能搜索。

- 结合营销中台丰富的数据资源，提供行业趋势、竞品分析等信息，帮助运营人员获取创意灵感，优化广告策略。

- **召回视频素材**

- 借助扣子编程强大的插件能力，墨攻团队成功接入自建的抖音多模态检索工具，实现了在抖音平台上对授权泛内容的高效召回。抖音作为拥有海量视频内容的平台，其丰富的素材资源为广告创意提供了广阔的素材库。

- 结合扣子编程工作流编排不同的搜索逻辑，系统能够精准提取用户意图，深入分析用户所处场景，并从用户输入的 Query 中获取相关参数。在此基础上实现关键词召回、画面相似召回或精确搜索等多种搜索方式，满足用户的多样化需求。

- **系统集成与优化**

- 依托营销中台前端组件，实现对话式交互模式，提升用户体验。

- 结合 ISP 中台提供的定制化扣子编程插件，系统完成了异常处理等关键逻辑，保证工作流的稳定性。

- 通过在系统中设置埋点，墨攻团队能够深入分析用户提问与系统回复之间的匹配度，持续优化检索和推荐策略，提升系统的精准性与实用性。

## 客户收益

通过扣子编程的智能化改造，墨攻团队在广告素材管理和创意生成方面实现了显著的效率提升。

- **查询效率提升**：在创意搜索和视频召回等场景下，双周使用量达 5000+ 次，大幅减少了人工找创意的时间成本。

- **自动化程度提高**：借助扣子编程的工作流，团队实现了广告创意数据的自动检索、用户意图的自动分析、场景参数的自动提取储，显著降低了人工干预需求。

- **工作流执行优化**：通过大模型 SQL 生成能力，简化了数据查询流程，使得复杂查询能够通过自然语言轻松实现，提升了操作便捷性。

- **产品迭代改进**：埋点数据提供了丰富的用户反馈信息，使团队能够及时发现回复的相关性、空回复率、错误率、响应时间等方面的优化空间。
